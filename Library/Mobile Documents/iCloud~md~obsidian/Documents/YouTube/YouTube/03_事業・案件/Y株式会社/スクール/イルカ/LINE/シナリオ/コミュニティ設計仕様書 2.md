# Discordコミュニティ設計仕様書（社内共有用）

- 1. コミュニティ構造と権限設計
    - 概要
        - 本コミュニティは「無料会員（労働力・見込み客）」と「有料会員（発注者・ツール利用者）」の階層構造を持つ。
    - 1-1. 会員種別と主な権限
        - 無料会員（一般）
            - ポジション：受注側・学習者
            - 採用活動：× 募集不可（応募のみ可）
            - バズミエ：△ 無料トライアル（7日間）
            - DM機能：× 原則不可
            - 参加条件：なし（誰でも参加可）
            - 発言権：△ 感想アンケート記入者のみ
        - 有料会員（VIP）
            - ポジション：発注側・実践者
            - 採用活動：◎ 募集可（無料/有料会員双方へ）
            - バズミエ：◎ 無制限利用（単体契約含む）
            - DM機能：◎ 応募受付用チャンネルあり
            - 参加条件：決済完了者
            - 発言権：◎ 最初からあり
    - 1-2. ランク制度
        - 目的
            - 有料会員のステータスを可視化し、無料会員への憧れと有料会員の継続率を高める。
        - ランク一覧
            - Lv.1：ビギナーディレクター
                - 条件：有料会員登録（収益化実績なし）
                - バッジ：🌱
            - Lv.2：プロディレクター
                - 条件：収益化達成済み
                - バッジ：⚡
            - Lv.3：エキスパートディレクター
                - 条件：月間収益50万円達成
                - バッジ：🔥
            - Lv.4：エリートディレクター
                - 条件：月間収益100万円達成
                - バッジ：💎
            - Lv.5：レジェンドディレクター
                - 条件：月間収益300万円達成
                - バッジ：👑
    - 1-3. 有料会員提供サービス一覧
        - 採用権あり
            - 無料会員プールを含むコミュニティ全体への求人募集が可能
        - 半永久アップデート講座内容動画
            - 15〜20分の講座が60〜70本ほど（随時追加・更新）
        - 受講生向けの非公開案件
            - 一般には出回らない高単価案件やクローズド案件の紹介
        - 受講生限定オフ会
            - 横の繋がりを作るリアルイベント参加権
        - グループチャット
            - 講師や他の受講生と交流・相談ができる専用チャット
        - 審査制チャンネル共同運営権限
            - 一定の基準を満たした会員限定で、運営側と共同でチャンネルを立ち上げる権利

- 2. チャンネル設計と運用ルール
    - 2-1. チャンネル構成と閲覧権限
        - 🔰はじめに
            - 役割：ルール・ランク説明
            - 閲覧権限：◎ 無料会員 / ◎ 有料会員
            - 詳細：ルール説明、ランク制度の説明
        - 📢運営からのお知らせ
            - 役割：LINE誘導・告知
            - 閲覧権限：◎ 無料会員 / ◎ 有料会員
            - 詳細：【重要】LINE誘導、期間限定キャンペーン告知、バズミエ告知
        - 🎁プレゼント受取
            - 役割：アンケート提出
            - 閲覧権限：◎ 無料会員 / ◎ 有料会員
            - 詳細：感想アンケート提出フォーム（提出で発言権付与）
        - 💼案件公募（求人板）
            - 役割：案件のマッチング
            - 閲覧権限：△ 無料会員（閲覧・応募のみ） / ◎ 有料会員（閲覧・応募・募集）
            - 詳細：投稿権限は有料会員のみ
        - 📈バズミエ速報
            - 役割：ツール威力チラ見せ
            - 閲覧権限：◎ 無料会員 / ◎ 有料会員
            - 施策：レビュー投稿でトライアル延長キャンペーン
        - 🏆成果報告
            - 役割：ランクアップ申請
            - 閲覧権限：◎ 無料会員 / ◎ 有料会員
            - 詳細：ランクアップ申請の場（承認制）
        - 🔥今日の積み上げ
            - 役割：作業報告・宣言
            - 閲覧権限：◎ 無料会員（投稿可） / ◎ 有料会員
            - 詳細：アンケート提出済みの無料会員も投稿可
        - 🔒バズミエ本番環境
            - 役割：ツールログインURL
            - 閲覧権限：× 無料会員（タイトルのみ） / ◎ 有料会員
        - 🔒リサーチ実践共有
            - 役割：生データ・ノウハウ
            - 閲覧権限：× 無料会員（タイトルのみ） / ◎ 有料会員
        - 🔒人材募集_投稿用
            - 役割：求人投稿フォーム
            - 閲覧権限：× 無料会員（タイトルのみ） / ◎ 有料会員
        - 🔒VIP雑談
            - 役割：上位層の交流
            - 閲覧権限：× 無料会員（タイトルのみ） / ◎ 有料会員
        - 補足
            - ※「無料会員が閲覧不可」のチャンネルは、無料会員がクリックすると「このチャンネルを見るには有料プランへのアップグレードが必要です」等のメッセージが出る（または鍵マークで見れない）設定とし、「そこに何かあるが見れない」ストレスを与える。
    - 2-2. バズミエ（ツール）の取り扱い
        - 販売形態
            - 単体販売あり（スクール必須ではない）
        - 無料戦略
            - 7日間無料トライアル（全機能開放）
        - 目的
            - 無料会員プールを「バズミエ単体販売」のリストとして活用
        - 追加特典
            - レビュー投稿でトライアル期間延長

- 3. セールス・導線設計
    - 3-1. オファーのタイミング
        - 常時：Discord内に固定リンク（LP）設置
        - 期間限定：LINE配信メイン（ローンチ形式）
    - 3-2. 決済・価格
        - 支払い方法：一括払い / 5回分割払い
        - 価格：15万円（決済会社） / 13万円（銀行振込）

- 4. 運用体制
    - 管理
        - 専任スタッフ配備
    - 荒らし対策
        - アンケート未提出者は発言不可（ROM専）

- 5. ウェビナー運用（ストリーム・カット方式）
    - 目的
        - 無料・有料・コンサル生の属性別に出し分けを行う
    - 属性別対応
        - 無料会員
            - 参加場所：YouTube Live（限定公開）
            - 配信内容：冒頭15分のみ視聴可能
            - アクション：15分経過後に配信停止し、有料への誘導スライドを表示
        - 有料会員
            - 参加場所：Zoomウェビナー（本会場）
            - 配信内容：全編視聴可能
        - コンサル生
            - 参加場所：Zoomウェビナー → 別Zoom
            - 配信内容：全編 ＋ 終了後のシークレット講義（Q&Aなど）
