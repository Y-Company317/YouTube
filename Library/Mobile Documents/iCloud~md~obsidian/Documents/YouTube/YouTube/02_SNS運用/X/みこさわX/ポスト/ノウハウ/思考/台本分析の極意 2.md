結局、YouTubeを当て続けている人は、
「分析」を徹底的に行っている。

「あの人の動画は面白いから伸びる」
「自分の動画は面白くないから伸びない」

解像度が抽象的なうちは、一生伸び悩みます。

解像度を上げるためには、
「伸びている動画」と「伸びていない動画」の差分を見つけること。


ただ動画を見るのではなく、以下の3つのステップで
伸びた理由/伸びない理由を言語化し、自分の台本に落とし込みます。

ステップ①：伸びている動画の共通点を洗い出す

まずは、同ジャンルで伸びている動画をピックアップし、以下の7項目でチェックします。

1. 冒頭のフック
   （冒頭30秒で「続きを見たい」と思わせる強烈な引きがあるか？）
2. 既視感の有無
   どこかで見たことある内容や編集ではないか？視聴者の需要を満たせているか？
3. 情報の新しさ
   動画内で納得させられる、新情報はあるか？
4. 感情の起伏
   感情が動くポイントはどこか？
5. 仮想敵の有無
   誰を敵に設定し、どのような視聴者を味方につけて共感を生んでいるか？
6. 伏線と回収
   視聴者を動画の最後まで見させる構成になっているか？
7. 情報量
   内容がスカスカで引き伸ばしていないか？

特に私が重視するのは「4. 感情の起伏」と「5. 仮想敵」


ステップ②：伸びていない動画の共通点を洗い出す


次に、「伸びていない動画」も見ます。
同じテーマなのに、なぜこちらの動画は再生されていないのか？

見るべきポイントはステップ②の7つの要素です。

これらをリストアップし、自分の動画では絶対に組み込まないようにメモをします。

失敗した動画の共通点を避ける方が、再現性は高くハードルが低いと考えています。

ステップ③：コメント欄の分析

ここが一番重要です。

多くの人が発信する内容ですが、「コメント欄」に視聴者の需要が詰まっています。

私は以下の4つの視点でコメントを分類し、自分の台本に反映させます。


1. ポジティブな共感コメント
   「わかる！」「これ知りたかった！」
   → 絶対に自分の台本にも組み込み、さらに強調すべきポイント

2. ネガティブな不満コメント
   「ここが分かりにくかった」「情報が古い」「声が聞き取りにくい」
   → 競合の弱点。自分が改善すれば、それだけで勝てる要素になる

3. 質問や疑問コメント
   「これってどういうこと？」「〇〇の場合は？」
   → 説明不足な点。丁寧に解説してあげるだけで「わかりやすい」と評価され、上位互換になれます

4. 補足情報や訂正コメント
   「実は〇〇なんですよ」「補足すると〜」
   → リサーチ不足の穴。この追加情報を自分の動画に組み込めば、情報の網羅性が上がる

特にネガティブなコメントを分析し、
「視聴者のストレス点」を解消した動画を出せば、後発でも勝てる動画を作ることができます。


「なぜ伸びたのか」
「なぜ伸びなかったのか」
「どういう動画が視聴者の需要を満たせるのか？」

この解像度を高める作業こそが、YouTube運営者の本質です。
