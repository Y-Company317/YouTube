# 質問
「月収30万円ほどで止まっている方を、月収100万円に伸ばすためのスクールを行っている」とお話しされていましたが、月30万円の方と月100万円に到達している方とでは、マインド面・テクニック面・才能面などにおいて、どのような違いがあると感じていらっしゃいますか？

# 回答
結果を出す人と遅い人の特徴

結果が早い人、大きく伸びる人、結果が遅い人の【差】はどこから生まれるのか？
◆結果を出す人の特徴
・腹くくっている。
・仕事として取り組んでいる。
・過去の努力量の中で一番現在やり込んでる。

結果を出す人は、どこかで「ここで結果を出さないとやばい」と本気で思っています。

この本気という部分が個々で基準やレベルが違いすぎる。

例えば、僕の場合は、美容師時代に一度自殺未遂しています。
借金苦で離婚も決まり、どうしようもなく自分から命をたった。

結果的に生きていますが、
それ以降、「こんな人生嫌だ」と心底思い、吐きながら生活していました。

そこから抜け出すためには自分が変わらないといけない。
今の現状は過去の自分が甘く招いた結果だからと気付いたからです。

そこから腹を括り、文字通り決死の覚悟で取り組んだ。

美容師時代は朝８時には出社。帰宅は２２時。
そこから朝５時まで動画作成。

身体的にもきついですし、精神的にもきつい。
だけどやるしかなかったですし、余裕がなかったわけです。

寝落ちするなどもなく、精神的な異常な状態。
寝るのが怖い。寝たら昔に戻る。

ベットを寝れない状態にして、
「この動画作成ができなければ死のう」
そんな覚悟で毎日行動していました。

結果的に、本気で取り組んで４ヶ月で月収１００万達成。
その年の１２月には月収１０００万達成。

本気で取り組んで８ヶ月後の出来事です。

この経験から全ての基準値が変わります。

「やればできる」
「やるとしたら異常値までやる」
「他者が面倒臭いことを即実践する」

そして本業になる前からですが、
美容師と同じ熱量で取り組んだことも大きな変化です。

このような話を伝えると、

「自分には無理だ、、、できない。」

と勝手に諦める方がいます。

やってもないのに、、、、
本気で取り組んでもないのに、、、、

やることが大変で、そこに本気で向き合うことが怖いから、
やらない理由をつけて行動しないわけです。

これが結果が出る人と出ない人の違いだと考えます。

次に

◆結果が早い人

教えている方を観察していると、

・即行動（リサーチ、アウトプット、動画作成）
・言われたことを素直に実践

このような方は非常に早いです。

逆にリサーチやアウトプットをサボる人ほど結果を出すのが遅い傾向です。

僕はコンサルをする上で、相手の思考レベルを紐解きながら、
質問形式でやりとりすることが多いです。

アウトプットできない＝考えてない

とニアリーイコールくらいに考えています。

忙しいからできない。
言語化が苦手だからしない。

いや、だからその結果なんでしょう、と心底思うわけです。

時間は作るもの。
苦手だからこそ、克服すればレベルアップできる。

なのに、覚悟が緩く、時間が無限にあると錯覚して行動しないわけです。

逆に結果が早い人は、
本気の覚悟を持って、時間が有限だと「正しく理解」しています。

みんな情報としては知っていても、
本気で理解していません。

今日生きていた人が、３秒後には死ぬかもしれません。
今日事故で死ぬ人は、朝起きた瞬間に「あぁ、俺は今日、事故で死ぬわ」とも思いません。

別れは突然やってきます。
世界で一日どれくらい人が生まれ、死んでいるかを調べてみてください。

その一人が自分であり、家族であり、仲間であるかもしれない。

これが事実です。
時間は有限で、僕らは偶然的にも生かされているわけです。

限られた時間の中で、
なぜ先延ばしするのか。

覚悟が緩く、余裕があるからです。

例えば、大事な人が来月手術でお金が足りない。
手術ができれば生きれる。やらなきゃ死ぬ。

そんな状況の時に寝落ちしますか？

しないですよね。

睡眠時間を削って、人に頭を下げてお金を借り、
売れるものを売ってお金を作りますよね。

それが本気で行動した結果です。
時間が有限で、行動しなければ大事な何かを失う。

これがリアルに見えたから思考や行動が変わったわけです。

そんな状況を作りましょう！というわけではなく、

いつかどこかで起こる未来なので、
「今」が最速なわけですから、すぐやりましょう。

ということです。

この今までお伝えしてきたことを、

・言われたことを素直に実践

できるかどうかです。

僕も普通の凡人です。
ただ、人が考える以上の努力はしています。

自分では努力と思わず、ゲームのレベル上げだと思っていますが。

「井上さんだから特別な考えなんでしょう。」

と思うか、

「よし！１００％の理解はまだできないけど、
素直に耳を傾けて、その心意気で毎日を生きよう！」

と思うかで見えるものや行動が変わってきます。

この素直さ非常に大事。

「でも」「だけど」を使わない。
言い訳をしない、。
できない理由を探さない。

これができれば結果が早いです。

◆大きく伸びる人
・異常値の基準値をもつ。（行動量、思考量、経験量）
・自分との中で比較（自分の過去と比べて成長）
・欲求に素直

短期間で異次元の数値を叩き出す人は、
・結果を出す人の特徴
・結果が早い人の特徴
これはベースとしてあります。

その中で異常値を叩き出す方は、

・行動量が異常
・考える深さが異常
・人生経験が異常

などの特徴があります。

僕の場合は、過去から分析すると

「人生経験が異常値」→「行動量が異常値」→「考える深さが異常値」

こんな感じで成長しました。

人生経験としては、

自殺未遂。
月に５万しか稼げてないのに背水の陣で美容師辞める。
お金稼いでみたら子供が障害持つ。
上手くいってると思ったら裏切られて数千万円の損失出したり。
直近で言えば、コロナ肺炎で人生二回目の物理的死を意識。

こんな感じです。

これを経験する必要はないですし、
僕はこんな星の元に生まれたんだと勝手に思っています。

なので、後天的にできるのは、

「行動量が異常値」
「考える量が異常値」

この辺です。

ビジネスに人生をかける覚悟なども異常値がある方もいますが、
それは各々でレベル差がありますし、正解がない世界です。

「行動量が異常値」
「考える量が異常値」

動画を作りまくる。
リサーチをしまくる。
分析をしまくる。
アウトプットしまくる。
インプットしまくる。

何かを異常なほど経験を積めば、
基準値が上がります。

一つだけでも異常な基準値を叩き出せば、
そこがベースになり、比較対象になります。

例えば、いつも吉牛の牛丼しか食べてない人間が、
一度でも良いからリッツカールトンの一杯五千円の牛丼を食べたとします。

この経験が異常値を叩き出す経験となり、
過去の比較対象として、その異常値と比べてどうなのか？を常に考えるようになります。

僕は常々、新しい経験、非日常体験をして新しい経験を積んでいきましょう。

と結果を出した方にはお伝えしています。

異常値の経験を色々な角度から積んでいけば、

他の行動や経験も、

「この経験の異常値はどこだろうか」
「どれくらいやりこめば異常値に行けるだろうか」

と考え出します。

この異常値の経験を積むためには、
精神的な苦痛や、肉体的な苦痛が伴うものです。
あとは金銭的な苦痛もあります。

そこを乗り越えてでも手に入れたいものがあるか。

それを自分の人生の中で、どれだけ本気で思っているのか。

この想いの部分も大事だったりします。

まとめると、大きな結果を出す人は、異常値の経験を積んでいる。
後天的に積むためには、自分から苦痛を伴うステージに喜んで進んでいく。

ちょっとした、頭のおかしさも必要です。クレイジーさですね。

一つの参考になれば幸いです。

最後に、

◆結果が遅い人

これは数を上がればキリがありません。

箇条書きにすると、

・素直じゃない
・行動しない
・他力本願
・面倒臭いを後回しにする
・自分にはできないと思い込んでる
・結果が出てる人は他者は特別だからと勝手に決めつけてる。
・考えることから逃げる
・言語化から逃げる。
・自分だったらこうするなど勝手にオリジナリティを入れ込む。
・拘りという名の変なプライドがある。

などです。

結局、自分はあまり変わらず、大きな結果を手にしたい！と
ちょっと頭がお花畑なことを思っています。（急に口が悪くなりすいません）

自分の人生なのに手を緩める。
ものすごーーーく甘い人間です。

結果が出ないことを他人や、環境のせいにする。
自分への負荷はかけないくせに新しい結果を求める。

筋トレしないのに、筋肉モリモリを求める。
告白もしないのに、美女からの告白を待っている。

こんな状態です。

いつか微量の努力の継続で、大きく結果が変わると信じているのかもしれません。

逆です。

一気に努力するから、一気に人生が変わる。

１年あれば人生を変えるには余裕な期間です。

人がしない努力。
人が面倒くさがって避けていること。
やらなきゃいけないけど後回しにしてること。

これらを愚直に行うからこそ、
人がいけない世界や、生活や人生の豊かさが手に入ります。

行動しない、考えることもしない、言われたこともしない。

できない理由を挙げていつまで経っても先延ばし。

そんな人がお金と時間を自由に手にするほど甘い世界ではありません。

逆に言えば、
人がしないほどの努力や、思考を常に行い言語化していく。

脳みそを活性化させていく。

これだけで人とは違う人生が歩めます。

僕は、高校卒業がギリギリでした。成績も下から２番目。
成績悪すぎて美容学校も落ちるくらいです。
１８歳から３４まで美容師しかしていません。

この仕事を選んで丸12年目に突入です。

後天的に手に入れた力ですが、本気で人生を歩んだ結果だと思います。
もちろん、切磋琢磨できる仲間たちや、恩師たちがいてくれたからこそです。

誰でも可能性がある世界なので、
今回の問題定義が人生を変える一つのきっかけになれば嬉しいです。
