# 台本プロット：高市首相×メローニ首相（Porta a Porta特集）

## 抑えるべきポイント

・文字数：8000〜9000文字

・視点設定：イタリアの人気政治番組『Porta a Porta（ドア・トゥ・ドア）』の女性記者ケリー・ジョンストン（Kerry Johnston）。
　→イタリアを愛する保守的な思考で、メローニ首相のファン。日本に対しては「経済は一流だが政治は三流」「リーダーが弱い」と批判的かつ厳しい目を持っている（視聴者意見の代弁）。

・オチの提示タイミング：高市外交の真価（メローニとの絆、アニメ外交）は後半まで見せない。前半は「また日本の首相がやらかすのではないか」という不安と緊張感を煽る。

・素材活用：実際の動画素材を積極的に組み込む（リンク参照）。

・時事ネタ：2025年〜2026年の外交スケジュール（G20、日中関係悪化など）を盛り込む。

・冒頭演出：OPで「日本が世界を混乱させた」かのようなCNN風の緊迫感ある演出を入れる。

## ストーリー全体の流れ（感情の波）

1. 不幸（日本の政治への絶望、メローニとの対比）
2. さらに不幸（G20での懸念、日中・日韓関係の悪化予測）
3. 少し希望（G20での高市×メローニの意気投合、マクロン攻略）
4. 絶望（中国の圧力、メローニ来日中止の危機）
5. 逆転（メローニ強行来日、アニメ・サンリオ外交）
6. 完全勝利（高市・メローニ称賛、主人公の価値観崩壊）

## 予定調和の裏切り（視聴者の予測 → 裏切り）

・視聴者の予測：「日本称賛動画だから、最初から日本が褒められるだろう」
→ 裏切り：OP〜序盤は「日本は世界の被害者になった」「日本の外交はあり得ない」と、まるで日本がトラブルメーカーであるかのように描く。主人公も日本に辛辣。

・視聴者の予測：「高市首相は強硬派だから、G20で孤立するだろう
（特にマクロンや中国、正直者のトランプやメローニとも対立するだろう）」
→ 裏切り：メローニと意気投合するだけでなく、犬猿の仲と思われたマクロンさえも味方につける。

・視聴者の予測：「中国の圧力でメローニ来日は中止になるだろう（政治的判断）」
→ 裏切り：政治的理屈を超えた「個人的な愛（アニメ好き）」と「友情」で圧力を跳ね除けて来日する。

## 構成詳細

### OP

・200文字以内。
・CNNが特集したかのような緊迫感ある演出。
・サムネ・タイトルの訴求（高市称賛）を裏切る入り出し。

・「日本のせいで世界が狂ってしまった」
「日本の外交は常識外れでマナー違反だ」
といったフレーズを使い、視聴者に「え？何をしたの？」と思わせる。

・実際は「良い意味で常識を破壊した」のだが、ここではネガティブなニュアンスでミスリードする。

・参考動画：https://www.youtube.com/watch?v=0XrW1b5Zc9c&t=1475s （14:45〜のような批判的な記者）

### 1. 不幸

・OP直後、緊張感・緊迫感のある様子で始める。
・イタリアの政治番組『Porta a Porta』のスタジオまたは取材現場。

・記者ケリーが、日本の歴代首相（岸田・石破）の外交を批判する。
（少し安倍首相の称賛を組み込む。トランプやプーチンと仲良くやっていたが、それ以外の首相で良い人はいない）
→参考動画3（https://www.youtube.com/watch?v=j7hGhHly4u4）や以下の動画のOP直後が参考
https://www.youtube.com/watch?v=0XrW1b5Zc9c&t=1475s

・メローニは左翼の人に厳しい。

・ケリーによる視聴者代弁（批判）：
　「前の首相（石破）を見てみなさい。自分を貶めるために総理になったのか？と思うほどだわ」
　「姿勢、食事マナー、表情、目つき…外交で最も大事な『品格』が見苦しい。党内で人望がない人間が、世界のトップに好かれるはずがないのよ」
　「短命に終わって本当によかった…と言いたいところだけど、次もどうせ同じような操り人形でしょ？」

・「安倍首相はトランプやプーチンと渡り合えたが、それ以外は見る影もない」と嘆く。

・メローニ首相の「国を想う強い姿勢」と比較し、「それに比べて日本は…」と呆れる。

・伏線：「外交は表面上の化かし合い。だが日本の政治家は化かすことすらできない」というケリーのセリフを入れる（後にブーメランとなる）

・参考：https://www.youtube.com/watch?v=bZEkAfWfx7A （0:55〜の「外交は化かし合い」のくだり）

### 2. さらに不幸

・2025年11月、南アフリカでのG20サミット直前。

・ケリーの懸念：
　①2025年6月のG7でマクロンとメローニが喧嘩（アイロール事件など）。今回も衝突するのでは？

　②新首相の高市早苗は「タカ派」で有名。中国・韓国とバチバチ（レーダー照射、レアアース問題）で、火に油を注ぐのでは？
　「この二人は違うと思う。メローニのように熱く語るタイプじゃない。でも、何を考えているか分からない不気味さがあるわ」と、高市の「静けさ」をネガティブに捉える。

・「メローニは正直者だから、嫌いな相手（石破など）には露骨に嫌な顔をする。高市に対してもそうなるのでは？」と、最悪の空気（カオス）を予想する。

・視聴者に「今回のG20は失敗する」という不安を植え付ける。

・G20開幕。ケリーの予想に反する展開。

・高市首相とメローニ首相が出会った瞬間、意気投合し、笑顔で会話（「表情外交」が良い方向に）

・さらに、マクロン大統領とも連携。

・メローニから「近いうちに日本に行くわ」という約束を取り付ける。

### 4. 絶望

・2025年12月〜2026年1月。事態が急変。

・中国が「日本を孤立させる」と宣言し、イタリアに猛烈な圧力をかける。

・【第一段階（子分レベル）】：中国外務省がイタリアに対し、ビザ発給停止と一部貿易制限を示唆。「メローニが訪日すれば、イタリアブランドの中国市場参入を禁止する」という脅し。

・ケリーの反応：「これは痛い…でもメローニなら耐えるかも？」

・【第二段階（ボス登場・絶望）】：習近平が直接動き、イタリアが依存するエネルギー供給ルート（一帯一路関連の残滓や中東ルート）に圧力をかける。さらに、EU内の親中派（ドイツの一部など）もイタリア批判に回らせる。

・構図の変化：「日本＆イタリア vs 中国」から、「日本 vs イタリア（中国・EU親中派に取り込まれる）」へ。

・イタリア国内でも「経済のために日本を切れ」というデモや圧力が高まり、メローニは孤立無援に。

・メディア報道：「メローニ、苦渋の決断。訪日を断念し、中国の顔を立ててオマーン訪問へ変更」

・ケリーの絶望：「終わりだ。あのメローニでさえ、中国という巨悪には屈した。日本は完全に独りぼっちになった」

### 5. 逆転

・2026年1月、メローニ電撃来日。

・ケリー驚愕：「なぜ？オマーンは？中国の圧力は？経済はどうするの？」

・【理由の深掘り・スカッと展開】
　実は高市首相とメローニ首相は、水面下で「ある作戦」を進めていた。

　①「オマーン訪問」は中国を油断させるための『囮（おとり）』だった。

　②高市首相は、中国が圧力をかけてくることを見越し、事前にG20でマクロンや他の欧州首脳、さらにトランプとも連携し、「イタリアへの経済制裁があれば、西側諸国全体で対抗措置を取る」という『裏の同盟』を固めていた（3章の伏線回収）

　③メローニの「個人的な愛（アニメ好き）」は、この強固な信頼関係の象徴であり、原動力。
　「私は北斗の拳を見て育った。愛のために戦うのが私の正義。脅しには屈しない」と、中国の圧力を一蹴して日本へ。

　④中国は「オマーンに行くと思っていたイタリアが、突然日本に現れ、しかもバックにトランプやマクロンがいる」という状況にパニック。手出しができなくなる。
　
・高市首相がサンリオグッズ（キティちゃん等）をプレゼントし、メローニが大喜びするシーン。

・「政治家である前に、一人の日本ファン」としてのメローニの笑顔と、それを守り抜いた高市首相の戦略的勝利。

・参考動画：https://www.youtube.com/watch?v=VqKwH1uS5aw （1:27〜プレゼント、4:22〜称賛）
・参考動画：https://www.youtube.com/watch?v=D9WCf9tkfnI （2:22〜来日の様子）

### 6. 完全勝利

・高市＆メローニの会談が大成功し、世界中が称賛。

・ケリーの独白（ブーメラン回収）：「私は『日本の政治家は化かすこともできない』と言った。訂正する。高市早苗は、正直さと愛で、外交の常識そのものを『破壊』して再構築したのだ」

・ケリーによる視聴者代弁（称賛とメディア批判）：
　「高市もメローニも、私利私欲に無関心で、純粋に国を愛している。そのイメージが本当に心地いい」
　「化かし合う外交は長続きしない。二人のように本心が伝わる外交こそが、真の国交よ」
　「高市はどんな相手にも誠意を持って接し、多くを語らず端的に話す。これぞ日本人特有の美しい所作ね」
　「でも驚いたわ。一国の首相が来日しているのに、日本のマスメディアは全く報道していないのよ」
　「日本では驚くほど報じられていない。だからこそ、世界からそっぽを向かれるのよ、日本のマスゴミは！」と、日本のオールドメディアを一刀両断して締める。

・中国や欧米も、この強固な日伊同盟（＋マクロン）を認めざるを得なくなる。

・最後は日本称賛で締めくくり。
