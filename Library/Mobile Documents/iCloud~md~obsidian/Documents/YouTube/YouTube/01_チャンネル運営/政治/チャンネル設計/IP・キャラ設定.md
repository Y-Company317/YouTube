# 政治チャンネル｜IP・キャラ設定メモ（カリスマ性・ファン化戦略）

目的：教祖様、ファン化、信者、カリスマ性を作るためのIP設計。  
香椎なつの「教祖感」と無敵おじさんの「エンタメ性」を掛け合わる。香椎なつに寄りすぎるとリスクになるため、回避しつつ熱狂的なファンを作る。

---

## IP設計ありのチャンネル設計で洗い出すべき項目一覧

**【チャンネル設計】**
- 視聴者理解と時代の変化（すべての起点）
- 視聴者に足りないもの（chの存在意義）
- 成功の3要素と3つのハードル（企画の判断フィルター）
- 価値提供と需要（誰に/何を/なぜこのchで）
- 視聴者の希望する未来（サムネの約束の源泉）
- 仮想敵と独自性（chの立場を決める）
- 企画とサムネイル（〇〇感フレーム）
- 競合の勝ち筋（構造を抜く）

**【IP設計：人格の設計】**
- MBTI：キャラの思考パターン・行動原理の土台
- 思想・価値観：何を信じていて、何を許せないのか
- 見た目：ビジュアルの印象（服装、髪型、体型、雰囲気）
- 口調：話し方のクセ、語尾、テンポ、丁寧さ/崩し具合
- 年齢：世代感、経験値の印象

**【IP設計：ギャップの設計】**
- キャラのギャップ：普段の姿と本気の瞬間の落差
- 口調と知識レベルの落差
- 見た目と中身の落差

**【IP設計：立場の設計】**
- 仮想敵：何と戦っている人なのか（人物ではなく構造・制度・環境を敵にする）

**【IP設計：見る理由の設計】**
- 誰がやっているのか：視聴者に見えるべき人格・スタンス
- なぜやっているのか：chを運営する動機・使命
- この人が歩む先の未来を見たい：長期目標。見届けたいと思える旅
- 次の投稿が気になる：連載的に追いかけたくなる仕掛け（予測→答え合わせ、伏線→回収）
- 成功も失敗も含めて追いたい：間違い・外れ・弱さの扱い方。完璧ではない人間味

**【IP設計：コンテンツとの接続】**
- 独自の視点と意見：このIPだからこその解釈・分析・切り口
- 視聴者にとっての「面白い」の定義：視聴者が快感を感じるポイント
- 掛け合い/役割分担：IPの魅力を引き出す相方やフォーマット
- お約束化できる要素：毎回繰り返すことでファンが定着する「この人らしさ」

## 今回の政治chの洗い出し結果

### 【チャンネル設計】

- **時代の変化による視聴者の需要の変化**
  - 3年前は「叩き/炎上/暴露」で伸びた→今は批判系に疲れている
  - 「善悪ジャッジ疲れ」が広がっている（裁判官をやらされる疲れ）
  - インフレ・増税の時代→「怒り」より「不安の解消（安心感/判断軸）」を求めている
  - TVへの不信→YTに中立で俯瞰的な情報を求めている
  - 「学んでいる自分」でいたい需要の増加

- **視聴者に足りないもの＝求めているもの**
  - 構造を整理して全体像を見せてくれるch（断片的な切り抜きchばかり）
  - 具体的な解決策や判断材料をくれるch（不安を煽るだけで「どうすればいいの？」がない）
  - 中立的な視点で俯瞰してくれるch（TVは偏向報道、YTは叩き系ばかり）

- **視聴者の成功の定義（見て良かったの3条件）**
  - 全体像の把握（時系列/因果/利害の整理で「結局こういうこと」がわかる）
  - 未来の見通し（この先どうなるかの予測と判断材料が手に入る）
  - 判断材料の提供（条件付き結論で自分の頭で判断できる）

- **視聴者の成功における3つのハードル**
  - 政治＝難しそう（専門用語が多い、ついていけない）
  - 解説ch＝ストレスになりそう（煽り/怒り/不安が増える）
  - 勉強＝面倒くさい（授業を受けさせられる感覚）

- **価値提供と需要**
  - 誰に：政治に不安はあるがTVは信用できず、自分で調べるのは面倒で、整理してくれるchが欲しい層
  - 何を：安心感と判断軸を持ち帰れる体験。「結局こういうことか」という腹落ち感
  - なぜこのchで：脱力した掛け合いで盗み聞きする感覚。気軽なのに最後は論点が整理される（IPがあるから成立する体験）

- **視聴者の希望する未来・目標・ゴール**
  - 短期：「今話題のあの件、結局どういうことなのか？」が安心してわかる
  - 長期：政治の全体像がわかる博識な自分になる / 判断軸を持って振り回されない / 家族や友人に「詳しいね」と言われる

- **仮想敵の設定＝独自性**
  - 仮想敵：増税する制度 / 煽りだけのch / 断定ch / 偏向TV / 難解専門家解説 / 解決策なしの批判ch / 断片切り抜きch
  - 独自性：ギャグ×お笑い / 掛け合い形式（盗み聞き感） / 失敗もギャグ化 / 減税ミッション・ゲージ / 構造ツッコミ / リッチ編集でTV特番感

- **企画とサムネ（〇〇感フレーム）**
  - コア感覚：安心感 / 腹落ち感 / 整理感 / 置いていかれない感
  - 伸ばす感覚：知的好奇心 / 裏側が見える感 / 見通し立つ感 / 語れるネタ化

- **競合の強み・弱み・勝ち筋**
  - BMの強み：テンポとギャグで報酬二重化、全方位いじりで中立に見える
  - BMの弱み：事実誤り/制度説明のミス、判断材料が弱い、IPの深さが足りない
  - 勝ち筋：一次情報/時系列を徹底 + 終盤で判断軸を回収 + IPを深く作り込む

- **視聴者にとっての「面白い」の定義**
  - 安心感 / 腹落ち感 / 網羅感 / 構造が見える感 / 代弁してくれた感 / 語れるネタ化 / 置いていかれない感 / 知的好奇心

  - 表の目標：若者の政治関心を高め、日本の未来を明るくする（学びの提供）
  - 裏の目標：減税を叶えて楽に生きるために政治をいじくり倒す（本音=視聴者の共感）
  - 煽りではなく整理で不安を消したい。偏向報道に振り回されない判断軸を届けたい

- **この人の理想の未来・目標**
  - 最終目標：減税（手取りを増やす）。達成されるまで動画投稿をやめない宣言
  - 減税ミッション/減税ゲージで進捗を見える化（長編ストーリー化）

- **次の投稿が気になる状態を作る設計**

  - コメント欄のA/B投票→次回冒頭で拾って参加価値を回収

- **成功も失敗も曝け出す**
  - 予測が外れたら：ギャグ化＋訂正（外れを隠さない＝信頼が積み上がる）
  - 当たった時：塾長が素直に喜ぶ＋シンが「珍しく正解」と淡々と褒める
  - 外れた時：塾長が開き直って脱力で誤魔化す＋シンが真顔で減点

  - 師弟関係（塾長＝師匠/シン＝弟子）：距離が近い師弟、扱いは雑だが見捨てない

- **シグネチャーフレーズ**

  - 「はい減点」（シンの折檻お約束）

  - ギャグ、お笑い要素

### キャラクターのスタンス：最強のトリックスター（脱力した賢者）
名前：**脱力塾長**  
MBTI：**ENFP**
- **「価値観の押し付け（上から目線）」ではなく「野次馬視点（高みの見物）」**
  - 政治家を「バカにする/見下す」要素は残すが、ブチギレて怒るのではなく **「呆れて笑う」「ギャグに持っていく」**。
  - 政治全体をネタの候補にする「コメディアン/芸能人」的な立場をとる。

- **見た目と行動のギャップ**
  - **外見**：一見それっぽい「塾長の制服」で、舐められない“格”を担保する
  - **行動**：コタツに入る、ソファでくつろぐ、寝転びながら喋るなど、**徹底的にリラックスした姿勢**で解説する
  - **効果**：「こんなにダラダラしているのに、言っていることは鋭い」というギャップが、「余裕のある強者」として映り、カリスマ性を高める

#### 運営者IP（「このチャンネルでなければいけない理由」）
狙い：視聴者が「情報」ではなく **“人（IP・運営者）” を追う理由**（＝次の投稿が気になる／成功も失敗も含めて見届けたい）

##### 1) 「このチャンネルでなければいけない理由」（唯一性）
- **脱力×論点整理の両立**：政治を“重い授業”にせず、脱力しながらも **論点だけを切り出して整理**できる
- **中立演出**：誰かの肩を持つのではなく、**政治批判を呆れた笑いに変換**しつつ、最後は視聴者に学びを届ける（感情→理解の二段報酬）。
- **師弟の掛け合いで担保する信頼**：塾長が脱線しても、弟子のシンが「根拠」「いつの話」「結論」を冷静に折檻して **軌道修正**する（＝置いていかれない感の強調＝安心感）

##### 2) 「このチャンネルだからこそできること」（提供価値）
- **ニュースの“再編集”**：出来事を煽りで終わらせず、**時系列・因果・利害**で再編集して「結局こういうこと」を作る。
- **判断軸を“条件付き”で渡す**：断定の押し付けではなく、**AならB（条件）**で提示し、視聴者が自分の頭で判断できる形にする。
- **コメント欄を“議論”ではなく“小テストの回答の場”にする**：最後に「A/Bどっち？」を投げ、次回冒頭で拾って **視聴者の参加価値**を回収する。

##### 3) 「一貫性の立場での意見」（ブレない軸＝ファン化の芯）
- **政党・人物ではなく“構造/制度/プロセス”を見る**：人格攻撃を避け、意思決定の仕組み・税制・制度の矛盾など **システム構造視点**で語る。
- **事実と意見と希望を分ける**：一次情報/根拠 → 解釈（論点）→ 条件付き結論（判断材料）の順で、毎回同じ型で話す。
- **執着ギャグの軸（例：減税）**：推し活ではなく「実益（手取り）」を軸にし、どの党でも“実現条件”で評価する（＝立場がブレない）

###### 減税（最終目標）を「最後まで見届けたい」に変えるキャラ設定
狙い：減税を“スローガン”で終わらせず、**進捗が追える長編ストーリー**にする（次回視聴の理由を作る）。
- **長期目標の宣言を固定**：「最終目標：減税（手取りを増やす）。達成されるまで動画投稿を赤字でも過酷さにのたれ死ぬまでやめない」（毎回OPで一言）
- **短期ミッション化**：毎回「今月の減税において確認すべきミッション・見ておくべきポイントや政策を1〜3個」を提示  
  例：法案の動き／発言の一貫性／財源の具体性／選挙後の行動…など“条件”で追う（推し活化を防ぐ）
- **進捗の見える化（お約束）**：動画内に「減税ゲージ/進捗メーター」を出す。**上がる条件や予想/下がる条件や予想**を明示する
（視聴者が発信者を“追う理由”を持てる）
- **師弟の役割分担**：
  - 塾長：希望（減税）を“執着ギャグ”として言い続け、熱を絶やさない（連載の旗）
  - シン：毎回「条件は満たした？」「いつまでに？」で折檻し、**現実チェック（信頼）**を担保
    - **期限**：いつまでに何が起きる想定？（会期/選挙/予算/成立時期など）
    - **プロセス**：実現までの手順はどこまで進んだ？（提出→審議→採決→成立 など“今どこ？”）
    - **数の根拠**：可決に必要な人数/合意は足りてる？（与野党/連立/党内の賛否）
    - **一貫性**：過去の発言・投票行動と矛盾してない？（ブレ/手のひら返しの検知）
    - **制度の制約**：法制度・憲法・省庁手続き・自治体など、構造的な制約に引っかからない？
    - **一次情報**：根拠は何？（公式資料/会見/法案/議事録など“ソース提示”を促す）
- **達成/未達を“イベント化”**：達成が近づくほど「Xデー」「最終局面」など章立てを作り、視聴者の期待を積み上げる

##### 4) その人（IP）を見る理由（ストーリー/未来を追いたくなる設計）
- **失敗もコンテンツ化（信用を積む）**：間違いがあれば「訂正/補足」を出し、**学習して前進する姿**を見せる（成長物語）
- **視聴者と一緒に“観測”する**：「次はここを見る（チェックポイント）」を提示し、次回で答え合わせすることで **連載感**を作る。
- **運営者の「予測」を毎回必ず組み込む（当たり/外れを報酬化）**

### 4-1. OPでIPを宣言（定型）
- 例：「今日も脱力で、論点だけを解説します」
- ねらい：キャラの“余裕（強者）”を毎回起動し、視聴者の期待値を安定させる

### 4-2. 姿勢は“脱力”（威厳は設定で担保）
- コタツ/ソファ/寝転びなど、徹底的にリラックスした姿勢（＝余裕）

### 4-3. コメント欄活性：A/B投票を「政党論」ではなく「判断軸」に変換する
難所：運営者の意見を出すほど、特定政党の応援/批判に見えやすい  
解決：**“党”ではなく“条件（判断軸）”を推す**

#### 台本の構成（台本テンプレ）
- **① 事実（一次情報）の提示**：今回の争点はこれ。決まっている事実はここまで。
- **② 論点の棚卸し（判断軸を3つ）**：「家計」「将来」「制度（持続性）」など“党から独立した軸”にする
- **③ 運営者の意見（条件付き）**：「私は“党”じゃなくて“条件”で選ぶ。条件Aが満たされるならA寄り、満たされないならB寄り」
- **④ 投票**：「あなたの結論はA/Bどっち？理由は一行で（条件で）」
- **⑤ 回収**：コメントの多かった条件を次回の冒頭で読み上げて“参加価値”を作る

#### 使える言い回し（党派に見せないための宣言）
- 「私は〇〇党のファンじゃなくて、『条件（政策/数字/手続き）』のファンです」
- 「推し政党の話じゃなくて、“この条件が満たされるなら賛成”の話をします」
- 「今日は“誰が正しいか”より、“どの判断軸で選ぶか”だけ持ち帰ってください」

#### A/B投票の設計例（質問文）
- 「あなたは A（短期で効く） / B（長期で効く） どっち重視？」
- 「A（スピード優先） / B（制度の安全優先） どっち？」
- 「A（減税を先） / B（財源の道筋を先） どっち？ ※“党”じゃなく“順番”の話」
- 「A（今すぐの生活） / B（将来世代） どっちを守るべき？」

#### NGになりやすい例（回避）
- 「〇〇党は終わってる/最高」など党名での評価断定
- 「支持者が〜」など相手陣営を丸ごと煽る（コメント欄が場外乱闘化しやすい）
