# 政治チャンネル｜サムネ設計メモ（〇〇感）

目的：視聴者がサムネを見た瞬間に「得られる感覚（〇〇感）」を言語化して、**再現性のある型**に落とす。

関連：`運用ガイドライン.md`（誤情報・名誉毀損・著作権などのリスク注意）
関連：`政治視聴者.md`（視聴者の「面白い」の定義｜詳細版） / `チャンネル設計_IP設計_マインドマップ.md`

## 視聴者の「面白い」の定義（〇〇感の土台）
サムネ・企画・台本すべてで「面白い」の認識がズレないための基準。
詳細は `政治視聴者.md` に記載。ここでは企画/サムネ設計で即使える形でまとめる。

| # | 面白いの定義 | 〇〇感への接続（サムネ/企画で狙う感覚） |
|---|---|---|
| 1 | **新たな知識を得られたとき** | 知的好奇心 / 霧が晴れる感 / 語れるネタ化できる感 |
| 2 | **悩みや不安が解消する** | 安心感 / 見通し立つ感 / 危機回避できる感 |
| 3 | **網羅感** | 安心感 / 整理感 / 要点わかる感 / 置いていかれない感 |
| 4 | **価値観が変わる（パラダイムシフト）** | 腹落ち感 / 判断軸もらえる感 / 人生変わる感 |
| 5 | **構造が見えるようになる** | 構造が見える感 / 腹落ち感 / 俯瞰できる感 |
| 6 | **孤独な価値観に共感される様子** | 代弁してくれた感 / 正義感 / 参加してる感 |
| 7 | **ワクワクする（知識武装している未来への高揚）** | 知的好奇心 / 優越感 / 語れるネタ化できる感 |
| 8 | **不協和の解消** | 代弁してくれた感 / 正義感 / 納得感 |

### チェック：企画/サムネが「面白い」の8定義のどれを狙っているか？
- 企画を立てる時、8つのうち**最低2つ**は明確に狙う
- サムネは狙う「面白い」の定義に対応する〇〇感ワードを使う
- 「笑い」は面白いの定義に含まれていない点に注意。笑いは"休憩点/ガス抜き"であり、面白いの本体ではない

## サムネで抑えるべきポイント
・サムネを見て一瞬で何が解説されるか？どんなテーマかがわかりやすい内容にする
→どんなテーマなのかはっきりしているため、何の話で何が得られるか？が明確で動画を見た未来への安心感につながある

・企画とサムネの安心感の理由と具体例
完全網羅されている：「完全解説」「全史」「〜史」などのワード
知識が無くても置いてかれない感&優しさ：「超わかりやすく」「神解説」 


## 政治系視聴者が求める「〇〇感」
ユーザー提示（重要）
- **【一番抽象概念】安心感（落とし系/炎上系のサムネだと、不安だけ煽られて安心感が出ない）**

- **人生変わる感（少し深掘りすると「博識になれる感」→「炎上系」「落とし系」のサムネが伸びない理由）**

- **要点わかる感**
- **俯瞰できる感**

## 使い方（型）
- サムネを「出来事」ではなく **“得られる感覚（〇〇感）”** で設計する
- 1枚で狙う〇〇感は基本 **1〜2個**（欲張るとボケる）
- タイトルとサムネで〇〇感を揃える（言葉の認識ズレ防止）

## 安心感/網羅感を作る“具体ワード”（ベンチマーク観察で頻出）
狙い：サムネ時点で「これを見れば分かる（振り回されない）」を作る。

### 1) 知識がなくても「置いていかれない」約束
堅い授業のような警戒心を解き、「自分でもわかる」と安心させる。
- **超わかりやすく / ゼロからわかる**
- **神解説 / 徹底図解**
- **〜とは？ / 誰でもわかる**

### 2) 1本で完結する「網羅感」の約束
あちこち調べる手間を省き、最短で全体像を掴みたい欲求に応える。
- **完全解説 / 完全網羅**
- **全史 / 〜の歴史**（ストーリーとして聞ける安心感）
- **総まとめ / 徹底比較**

### 3) 結論が見える「スッキリ感」の約束
- **〜の末路 / 〜の正体**
- **崩壊 / 終焉 / 激変**（オチが明確）

## メモ：腹落ち感（霧が晴れる）を作る“問い/構造ワード”
- **なぜ〜なのか**：構造が見える感 / 推理できた感 / 腹落ち期待
- **崩壊 / 歴史 / 時代を振り返る**：因果・時系列でつながる期待（ストーリーで腹落ち）
- **天災と人災**：原因の分解（構造化）期待 / 善悪断罪より“理解”へ寄せやすい
- **どんな人か**：人物理解（語れるネタ化）/ 知的好奇心

## メモ：「人生チラ見え」系で作る〇〇感（人物回で強い）
- **壮絶すぎる人生**：人生の“チラ見え”で好奇心を喚起しつつ、「運営者が完全に理解している感」（＝信頼できる感/網羅感）を作りやすい

## メモ：なぜ視聴者は「高市さんがどんな人生（経歴）を送ってきたか」を知りたいのか？
前提：政治視聴者は「何が本当か分からない」「判断軸が欲しい」「言動不一致（偽善）に強く反応する」傾向がある。  
→ だから人物回の“人生/経歴”は、単なるゴシップではなく **今後の総理大臣を知り、未来がどうなるか知りたい or 未来の不安を安心するための判断材料（安心）** として需要が出やすい。

### 1) 網羅感につながる理由（断片情報→一本の“時系列”で統合できる）
- 視聴者はニュースや切り抜きで人物を断片的に知っている状態が多い  
  → 経歴を **時系列（起点→転機→現在）** でまとめると、情報が「一覧化」されて **全体像が掴める＝網羅感** が出る
- 人物の“過去の選択”や“立ち位置の変化”がつながると、政策や発言が「点」ではなく「線」になり、理解が一気に進む

### 2) 安心感につながる理由（人物が“信頼"できるか知りたい。未来のスクリーニングと行動予測ができる）
- 政治は個人で止められない領域で、視聴者は「振り回されたくない」  
  → 人物の経歴は「この人は何を優先してきたか」「逆風でどう動いたか」という **パターン** を見せる  
  → 未来の判断（支持/不支持ではなく“条件付きで期待できるか”）に使える＝**安心感/未来の見通し立つ感**
- 視聴者は「偽善/言動不一致」に強く反応しやすい  
  → 経歴で **一貫性（筋が通るか）** を確認できると「信頼できる感」「根拠ある感」が増える

### 3) スッキリ感（整理感）につながる理由（モヤモヤの正体が言語化される）
- 「なんか支持されてる/嫌われてるのは分かるけど、結局どんな人？」というモヤモヤが、経歴整理で解像度が上がる  
  → 視聴後に「結局こういうこと」が手に入る＝**整理感/腹落ち感/スッキリ感**

### 4) サムネ設計への落とし込み（網羅/安心/スッキリを同時に約束する）
- **網羅感**：`完全解説` / `全史` / `経歴ぜんぶ` / `時系列で整理`
- **安心感（判断材料）**：`何者？` / `原点` / `なぜ強い` / `一貫性` / `逆風で何をした`
- **スッキリ感**：`結局こういう人` / `3点で分かる` / `これだけ見ればOK`

## メモ：コンセプト文言が生む安心感（例：「一番面白い解説」）
- 「面白い & 分かりやすい解説」＝深く理解していないと成立しない  
  → 視聴者は「このチャンネルを見ると **博識になれる**」「**楽しみながら** 理解できる」「**時間を無駄にしない**」という安心感を得やすい

## 注意：疲れるワード（使うなら“学び回収”の約束をセット）
- **闇 / 裏側 / 炎上 / 失敗**：短期のスカッと/怒りは出るが、回収（整理→判断材料）が弱いと不安増幅で疲れる  
  - セット推奨：「仕組みを図解」「結局こういうこと」「見るべき3点」「完全解説」など

## メモ：サムネの「リッチさ」を作る/壊す要素（共通チェック）

### なぜ「リッチさ（エンタメ感）」が必要なのか？
政治などの「堅い/面倒」なテーマを、**「これは勉強ではなく、映画や特番のようなエンタメです」とパッケージング（偽装）するため**。
- **リッチ系（TV特番風）**：
  - **役割**：「重要感」「没入感」を出す。
  - **心理**：「手間がかかっている＝見る価値がある重大な情報だ」と信頼させ、学習の警戒心を解く。
- **まとめ系（週刊誌/スポーツ新聞風）**：
  - **役割**：「手軽さ」「覗き見感」「本音感」を出す。
  - **心理**：「教科書ではなく裏話だ」と思わせ、学習のハードルを下げる。


- **1) テレビの既視感の利用**
  - **ターゲット特性**：50〜60代は「テレビ世代」。金銀テロップや立体文字は、彼らにとって「ニュース特番/討論番組」の象徴。
  - **効果**：「テレビと同じクオリティ＝信頼できる情報だ」と無意識に誤認させ、**「怪しい個人チャンネル」という警戒心を解除**する。
  
- **2) 「気軽に学習モード」へスイッチする**
  - **教科書的デザイン**：「勉強しなきゃ（能動的）」と身構えさせる（疲れる）。
  - **リッチデザイン**：「ソファでテレビを見る感覚（受動的）」でクリックさせる（疲れない）。

- **3) 「手間がかかっているサムネ＝高品質」が安心感に繋がる**
  - 「サムネにお金/手間がかかっている」＝「中身のリサーチもしっかりしているはず」という**品質保証（安心感）**として機能する。

### 伸びるサムネの要素メモ：稲妻（発光/電撃）演出
- 稲妻が与える印象：「緊急性×重要性×対立（衝突）」  
  - 緊急速報感：「今起きてる」「見逃すと損/危険」発光は「特番/緊急」の演出
  - 重大さ（特番感）：テレビのニュース特番っぽい“格”と“事件性”
  - 衝突・分断のドラマ：党内抗争／選挙の勝敗／スキャンダルなど「ぶつかってる」空気
  - 感情の点火（興奮）：怒り・驚き・期待を一瞬で上げる（ギラギラ感）
  - リッチさ（制作コスト感）：背景に情報量が増え「作り込んでる＝中身も濃そう」に見える

- 「政治は“重要そうに見えないと”クリックされない」  
  →ここが非常に重要なポイント。メモしたい損失回避を強調したい。

- 画面設計として強い  
  稲妻は背景を割って奥行きを作れ、人物や文字の視認性（即読）も上げやすい

### リッチに見える要素（〇〇感）
- **即読感**：0.5〜1秒で「コンテンツ内容」と「どんなベネフィットがあるか（完全網羅など）」がわかる（迷わない安心感）
- **主役感**：視線が最初に刺さる焦点がある（人物/物/単語）
- **重要感**：コントラスト/アクセント色で“重大そう”が一発で伝わる
- **信頼感**：フォント・縁取り・余白・整列が揃っていて“丁寧に作っている”と感じる

### 安っぽく見えやすいNG（直すと改善が早い）
- **小さい要素のコラージュが多い**（主役不在で迷う）
- **文字が多い/弱い**（強弱がなく読めない）
- **コントラスト不足**（薄い色×薄い文字で沈む）
- **フォント/加工がバラバラ**（量産感が出る）

### 改善手順（テンプレ）
1. 色を **3色までに絞る**（ベース黒白＋アクセント赤or黄＋補助1色）
2. 太字ゴシック＋縁取り＋影で可読性を上げる

## 世界史/世界情勢×知的好奇心（world_history系）で刺さる〇〇感
一次情報：`https://www.youtube.com/@world_history/videos`

### 取りに行ける〇〇感
- **知的好奇心（なぜ？を解きたい感）**
- **腹落ち感（原因・背景がつながる）**
- **置いていかれない感（難語排除/カジュアル）**
- **スケール感（数字で重要度が一発）**
- **覗き見感（タブー/逆転テーマを“学び”として安全に見られる）**
- **重要感（見ないと損しそう感）**：赤/黄/黒など危険色・警戒色で「重大そう」を即伝達
- **スッキリ感**：ワードを短く切ることで「すぐ分かる」を約束
- **スッキリ感（整理される感）**：余計な説明を省き、見れば腑に落ちる期待を作る
- **霧が晴れる感（謎解き完了感）**：暗め・靄っぽい画作り→“視聴で解像度が上がる”期待感

### よく出る“問い/構造ワード”例
- **なぜ〜なのか**
- **実態**
- **〜の歴史**
- **すべてのはじまり（起源）**
- **〜年の因縁**
- **ゼロからわかる / 世界一わかりやすい**

### 衝撃・意外性で引っ掛ける“逆転ワード”例（使うなら必ず腹落ち回収）
- **今日から○○は合法**
- **たった1人で○○万人**
- **絶滅 / 地獄 / 史上最大**

### メモ：視聴者の認識ズレが起きやすい点（注意）
- **重要感（危険色）**は、視聴者によっては **煽り感/不安増幅** にも見える  
  - 回避策：タイトル/冒頭で「今日は“なぜ”を整理する」「最後に判断材料を渡す」と約束し、回収する
- **暗め＝霧がかってる感**は刺さる一方、強すぎると **怖い/重い/見てて疲れる感** に振れる  
  - 回避策：サムネの“闇っぽさ”より「解ける約束（なぜ/実態/ゼロから）」を強く出す
- **簡潔さ**は強みだが、短すぎると **浅い/雑** と感じる層もいる  
  - 回避策：サムネは短く、動画内で「時系列・因果・要点3つ」で厚みを出す

## ケーススタディ：なぜ「投票先」は伸びて「戦争」は伸び悩んだのか？（第二ベンチマーク分析）
対象：政治経済たぬじろう（@seiji-tanuki）

### 1) 伸びた企画：「どこに投票すればいい？」
- **〇〇感（視聴報酬）**：**判断軸もらえる感 / 迷いが晴れる感 / 損しない感**
- **自分事化の距離**：**「極小」**（選挙は全員に関係し、投票行動という“明日やるべきこと”に直結する）
- **サムネ設計**：
  - **全リーダーの顔出し** → 「ここを見れば**比較・網羅**できる」という約束（あちこち見なくて済む**時短感**）
  - **問いが具体的** → 「どこ？」という問いは、答え（＝得られるもの）が明確。
- **勝因**：視聴者の「分からない/迷う」という**具体的な痛み（課題）**を解決する提案になっている。

### 2) 伸び悩んだ企画：「戦争が起こるって本当？」
- **〇〇感（視聴報酬）**：**危機感（煽り） / 不安感**（※解決策が見えにくい）
- **自分事化の距離**：**「遠い」**（憲法改正や戦争は、生活実感として“今すぐ”の影響が見えにくい）
- **サムネ設計**：
  - **問いが抽象的/YesNo** → 「本当？」という問いは、答えが「本当/嘘」の二択になりがちで、**「見てどうなるか（実益）」**が想像しにくい。
  - **煽りへの疲れ** → 「戦争煽り」はYouTubeに溢れており、視聴者が食傷気味（またこの手の話か…という**既視感/警戒心**）。
- **敗因**：「不安」だけを提示しており、視聴者が**「見ることで得られるプラスの変化（安心/解決）」**を直感できなかった。

### 結論：自チャンネルへの転用ルール
1. **「不安」より「解決（判断軸）」を売る**：ただ脅すのではなく、「どうすればいいか」の答えをサムネで約束する。
2. **「遠い危機」より「近い実益」に翻訳する**：
   - ×「憲法改正の是非」
   - ○「あなたの徴兵リスクと税金はどうなる？（生活への翻訳）」
3. **「比較・網羅」は強い**：迷っている視聴者に「一覧表」を見せる企画は、ハズレが少ない（時短・整理の価値）。

## 企画の方向性/Goog & NG事例集
※ 企画の判断基準を統一するためのリスト

### 【良い例】
基本方針：「不安」を「解決/判断軸」に変える企画。自分事化の距離が近い、または知的欲求（網羅・構造理解）を満たすもの。

#### 1. 「迷い」を解決する企画（網羅感・実益性・安心感）
- **狙い**：情報過多で混乱している視聴者に「ここを見れば正解（安心感）が分かる」と約束する。
- **〇〇感**：判断軸もらえる感 / 迷いが晴れる感 / 時短感
- **事例**：
  - 「どこに投票すれば良い？」全政党網羅・比較

## 補足：知的優位性と実益を強める「企画・サムネ変換」
視聴者の「賢い理想の自己像」と「実益（危機回避）」に合わせるための微調整をする

### 1. ツッコミ、弄り要素を強調する企画、サムネ事例
単なる「馬鹿にする（人格攻撃）」は視聴者の品位を下げる。
「仕組みがおかしい」「計算が合わない」という**構造の矛盾**を指摘することで、知的な笑い（優越感）に変える。

- **企画の切り口**
  - × NG：〇〇議員の失言集（ただの悪口）
  - ○ Good：**「なぜその政策は100%失敗するのか？」**（仕組みの欠陥を指摘）
  - ○ Good：**「なぜ不可能なのか？」**（矛盾を笑う）

- **サムネの訴求ワード（知的エンタメ感）**
  - **「矛盾」「欠陥」「詰み」「無理ゲー」「バグ」「ブーメラン」**
  - **「算数ができない」「物理的に不可能」「崩壊の数学的根拠」**

### 2. 予言ではなく、「AになったらBになる（から備えよう）」という実益になる訴求をする
「当たるか外れるか（オカルト）」ではなく、「AになればBになる」という**確定した未来への備え**として提示する。

- **企画の切り口**
  - × NG：2025年、日本滅亡！？（煽り・オカルト・遠い）
  - ○ Good：**「【試算】この政権が続くと、あなたの手取りはいくら減る？」**（計算・実益）
  - ○ Good：**「高市早苗の『10年計画表（ロードマップ）』を完全解説」**（見通し・計画）

- **サムネの訴求ワード（危機回避・実益感）**
  - **「試算」「シミュレーション」「確定」「ロードマップ（計画表）」**
  - **「Xデー」「タイムリミット」「逃げ道」「備え」「処方箋」**
