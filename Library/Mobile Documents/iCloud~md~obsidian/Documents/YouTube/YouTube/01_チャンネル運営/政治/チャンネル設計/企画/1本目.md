## 競合分析（起点）

### 競合サムネ（第一ベンチマーク：政治無敵おじ）
- **動画リンク**：`https://www.youtube.com/watch?v=P6T9oekCGjw`
- **再生数**：69万再生
- **サムネ構成**：
  - メインワード：**「激ヤバ日中関係」**（赤背景・極太フォント・画面下半分を占拠）
  - サブワード：**「-高市内閣の現状-」**（中央帯）
  - 人物配置：高市早苗（左・大）、習近平（右・大）、トランプ（右上・中）、小泉進次郎（左上・小）
  - 背景：稲妻・国旗（日本・中国・アメリカ）・戦闘機
  - 色調：赤×黒×金＝緊急感・重要感・対立感
- **サムネの〇〇感**：重要感 / 緊急感 / 覗き見感 / 構造が見える感 / 置いていかれない感

### この企画が69万再生を取れた理由（仮説）
1. **複数トピックを1本に詰め込む構造**：高市内閣の施策＋日中関係＋トランプ外交＋衆院選の結果…と、1本で複数の「知りたい」を回収できるため、**網羅感・安心感**が強い
2. **時事性が極めて高い**：高市内閣発足後の「今どうなっている？」という現在進行形の関心に直結
3. **自分事化の距離が近い**：日中関係＝経済・安全保障＝生活に直結する不安（近い危機）
4. **人物が即読できる**：高市・習近平・トランプという「誰でも知っている顔」がサムネにあるため、0.5秒で何の話か分かる
5. **リッチさ（TV特番感）**：稲妻・国旗・戦闘機の背景で「重大ニュース特番」の信頼感を演出

### この企画の最大の強み＝「1本で複数のサムネ訴求ができる」構造
動画内でさまざまなトピックを解説するため、**サムネ/タイトルの切り口を何パターンも試せる**。
これが1本目に最適な理由：初動でどの訴求が刺さるかをABテストできる。

## 1本目の企画方針

### コンセプト
**「高市内閣のすべてを1本で完全解説する動画」**

高市早苗の人物史 → 衆院選の圧勝劇 → 高市内閣の施策 → 日中関係・外交の現状 → 今後の計画（ロードマップ）
…を時系列で網羅的に解説し、**1本で「高市内閣の全体像」が掴める**動画にする。

### なぜ1本目にこの企画か
- **チャンネル設計の「伸びる型」に完全一致**：全史/人物回/未来の備え/ツッコミ の4型すべてを組み込める
- **視聴者の〇〇感をフルセットで回収**：安心感 / 網羅感 / 腹落ち感 / 見通し立つ感 / 語れるネタ化
- **IPのお披露目に最適**：脱力塾長×シンの掛け合いを見せるには、情報量が多く構造整理が必要な長尺テーマが合う
- **減税ミッションの起点**：高市内閣の経済政策（減税/増税）に触れることで、チャンネルの縦軸「減税」を自然に宣言できる
- **サムネ差し替えで複数回チャンスがある**：1本の動画で4〜5パターンのサムネを試せる

### 追加：問いかけ（？）型サムネの訴求テンプレ（疑問→解決の約束）
「？」は**視聴者の“今あるモヤモヤ”を言語化して、クリックの理由（＝解決）を作る**型。  
※ 重要：問いは**具体**にして、サムネ上で「見ればどうなるか（報酬）」まで約束する（不安煽り単体はNG）。

#### 問いかけ（？）で取りに行ける訴求（強い順）
- **迷いの解消（判断軸）**：`あなたは何を基準に決める？`／`結局、あなたは何を見ればいい？` → 「迷いが減る」安心
- **比較・網羅（時短）**：`あなたの選択肢、何が違う？`／`結局どこが得？` → 「あなたのための早見表」安心
- **自分ごと化（行動直結）**：`あなたの手取り、どうなる？`／`次の投票、あなたはどうする？` → 「生活/投票に直結」
- **置いていかれない（入門）**：`ゼロからでも分かる？`／`いま、あなたの生活に何が起きてる？` → 「置いていかれない」安心
- **腹落ち（構造理解）**：`なぜ“あなたが損する構造”になってる？` → 「理由がつながる＝語れる」博識

#### 目的別：サムネ文型（自分ごと化 / 安心感 / 博識 / 網羅感）
- **自分ごと化できる（距離を“極小”にする）**
  - **文型**：`あなたの手取りは？` / `あなたの税金は？` / `あなたの生活、結局どっちが得？` / `次の投票、あなたはどうする？`
  - **具体化のコツ**：主語を「日本」→「あなたの家計/老後/安全/子ども/仕事」に翻訳（“財布 or 安全 or 家族”のどれかに寄せる）
  - **サブ約束（小さく添える）**：`あなたの判断基準3つ` / `損しない見方` / `見るべきポイント3つ`
- **安心感がある（見れば“迷いが減る”を約束）**
  - **文型**：`あなたは結局どうすればいい？` / `あなたは何を見ればOK？` / `迷ってるあなたへ`
  - **約束ワード**：`これだけ見ればOK` / `結論→理由→判断基準` / `3つだけ` / `ゼロから` / `チェックリスト`
  - **不安回避**：危険ワードを使うなら、必ず `整理して判断材料（条件）を渡す` を同居させる
- **この動画だけで博識になれる（語れる/説明できる未来）**
  - **文型**：`家族に1分で説明できる？` / `それ、あなたは言える？` / `なぜ？をあなたの言葉で語れる？`
  - **約束ワード**：`背景まで理解` / `構造でわかる` / `歴史からつなぐ` / `要点3つで語れる` / `一枚で説明`
- **網羅できる感（調べ疲れを終わらせる）**
  - **文型**：`あなたのための早見表：結局どこ？` / `あなたの選択肢、何が違う？（比較表）` / `全○○まとめ（保存用）`
  - **約束ワード**：`完全解説` / `完全網羅` / `総まとめ` / `徹底比較` / `保存版` / `早見表` / `比較表`

#### 問いかけ型のNG（伸びづらい/疲れる）
- **抽象的なYes/No問い**：`本当？` `ヤバい？` だけだと「見てどうなるか」が薄く、不安だけが残りやすい  
  → 置換：`結局どうなる？（生活への翻訳）` / `何を見ればいい？（判断材料）` / `なぜ？（構造）`
- **煽りの既視感**：戦争/崩壊/終焉だけで釣ると食傷されやすい  
  → セット：`図解` `結局こういうこと` `観測ポイント3つ` `処方箋`

## 運用ガイドラインチェック（1本目用）

### 守るべき最重要3点
- [ ] **誤情報を出さない**：出典/日付/時系列/仮説線引き
- [ ] **党派化しない**：高市称賛に偏りすぎず、構造主語で是々非々
- [ ] **学び回収で終わる**：整理→判断材料→観測ポイント

### 1本目特有の注意
- [ ] 高市さんは「称賛寄り」の対象（視聴者データ参照）→ 落としに寄せず「継続→学習→変化」で回収
- [ ] 自民の圧勝を扱う際、野党批判に見えすぎないよう構造主語で
- [ ] 日中関係は「遠い危機の煽り」にならないよう、「自分の生活への影響」に翻訳する
- [ ] 1本目なのでIPの魅力（脱力塾長×シンのギャップ）を意識的に見せる
- [ ] 減税ミッションの宣言で「特定政党の信者ではなく、政策（減税）の支持者」を明確にする
