- SEOで狙うべきなのは、単なるツール名ではなく「視聴者がずっと悩み続けること」
- 今回の検索意図は、`YouTubeリサーチツール` そのものを知りたい人だけでなく、`YouTubeリサーチ` `YouTubeリサーチ方法` `YouTube競合リサーチ` `YouTubeキーワードリサーチ` のように、リサーチ全般に悩んでいる人も含む
- つまり、`ツール名訴求` より `悩み解決訴求` の方が長期再生を取りやすい

## YouTubeリサーチツールを欲している人の主な悩み

### 企画が当たらない
- 何をテーマに動画を作ればいいかわからない
- 再生されるネタと再生されないネタの違いがわからない
- 思いつきで投稿してしまい、継続的に伸びない

### 競合分析ができない
- 伸びているチャンネルの共通点が見えない
- タイトル、サムネ、テーマ、切り口のどこを真似すべきかわからない
- ライバルが強いジャンルを避けたい

### キーワード選定ができない
- 視聴者が何を検索しているかわからない
- SEOを狙えるタイトルの作り方がわからない
- ロングテールで動画テーマを作れない

### リサーチに時間がかかる
- 毎回YouTubeを見回って終わる
- 情報が散らばっていて整理できない
- 調べたはずなのに次回また同じことを調べてしまう

### 伸びる型が見つからない
- どのジャンルなら勝てるか判断できない
- ショート向きか長尺向きか見極められない
- 今だけではなく、長期で再生されるテーマを探したい

### 数字の見方がわからない
- 再生数だけ見て判断してよいのかわからない
- 投稿頻度、再生速度、競合数など何を重視すべきかわからない
- 収益化につながるテーマを先に見つけたい

## フロント動画の企画方針

### フロント動画の役割
- YouTubeではまず `リサーチの重要性` を理解させる
- 視聴者に `伸びないのはセンス不足ではなくリサーチ不足かもしれない` と気づかせる
- `手作業リサーチの限界` を見せる
- 最後に `実際に使っているリサーチツール` の存在を伝えてLINEへ誘導する

### フロント訴求の結論
- 入口では `悩み` と `結果` を見せる
- 冒頭ではSEOのために `リサーチツール` という単語を一度入れる
- ただし、ツールの中身や詳細は全部話さず、最後まで奥行きを残す
- つまり `単語は出すが、答えは出し切らない` がベスト

## フロント動画で狙う構成

### 1. 冒頭
- `YouTubeで伸びない人の多くは、作る前のリサーチで負けています`
- `今日はYouTubeリサーチのやり方と、最後に僕が実際に使っているリサーチツールについてもお話しします`

### 2. 問題提起
- 伸びない人ほど編集や投稿本数を頑張っている
- でも本当に差がつくのは企画の前段階
- 感覚で投稿すると外し続ける

### 3. 気づき
- YouTubeは `頑張った動画が伸びる` のではなく `需要と合った動画が伸びる`
- その需要を見極めるのがリサーチ
- だからリサーチ不足は、努力不足より危ない

### 4. 手作業リサーチの限界
- 競合を見る
- 伸びた動画を見る
- タイトルやテーマを比較する
- メモを取る
- これ自体は正しいが、毎回やるには時間がかかる
- 抜け漏れも起きやすい

### 5. 解決の方向性
- だから、リサーチは仕組み化や効率化が必要
- ここで `自分が実際に使っているリサーチツール` の存在を示唆する

### 6. CTA
- `この動画の最後でも少し触れましたが、僕が実際に使っているYouTubeリサーチツールはLINEで詳しく案内しています`
- `気になる方は下のリンクから確認してください`

## 今回の企画で優先して狙うSEOワード

### 前提
- 今回は `視聴者の悩み` と `共有済みSEOデータの検索vol` をもとに判断する
- `YouTube 伸ばし方` は悩みとの相性は良いが、今回共有されたデータ内では検索volを確認できていない
- そのため、まずは `検索volが確認できていて、かつ悩みに直結するワード` を優先する

### 最優先で狙うワード
- `YouTube リサーチツール` : 月間検索数110
- `YouTube リサーチ` : 月間検索数90
- `YouTube リサーチ方法` : 月間検索数70
- `YouTube リサーチ 方法` : 月間検索数70

### 補助的に狙うワード
- `YouTube リサーチ やり方` : 検索volは0だが、タイトル・説明欄では自然に入れやすい
- `YouTube 競合 リサーチ` : 検索volは0だが、悩みとの一致度は高い
- `YouTube キーワード リサーチ` : 検索volは0だが、訴求として強い
- `YouTube 企画 リサーチ` : 検索volは0だが、企画に悩む人へ刺さる
- `YouTube リサーチツール 無料` : 検索volは0だが、LINE導線やCTAでは使いやすい
