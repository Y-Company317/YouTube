# この動画を出す目的
・チャンネル設計のコンセプトとして、YouTube関連のSEOキーワードをとり、長期的に伸び続けるチャンネルを作っていきたい

・この動画のゴール（KGI）は、講座の購入の促進です。
・その中のKPIとして、SEOをとるということがあります。
・コレは教育動画に近く、視聴者の中でも初心者層、入り口としてパイを広める戦略にも近いです。


# 競合の状況
YouTube始め方と検索すると、「やさしい始め方」「超初心者入門」「チャンネルの作り方」などかなり初心者向けの内容の動画が多くヒットします。

# ターゲット
また、視聴者層は以下のように9種類に分かれています。
この中でも、初心者と中級者初心者、 中級者の中の中級者ぐらいに刺しにいきたいです。

 - 初心者
    - 完全初心者（まだ副業を始めていない人）
    - 初心者の中の中級者（YouTubeに少しだけ取り組んだことがある人）
    - 初心者の中の上級者（YouTube開始後1ヶ月程度の人）
 - 中級者
    - 中級者初心者（YouTube開始後2〜3ヶ月の人）
    - 中級者の中の中級者（YouTube開始後半年程度の人）
    - 中級者の中の上級者（YouTube開始後1年程度の人）
 - 上級者
    - 上級者の中の初心者（YouTubeで一定の収益を安定的に得ている人）
    - 上級者の中の中級者（YouTube収入が本業並みかそれ以上の人）
    - 上級者の中の上級者（YouTubeで大きな成果を上げ、さらに拡大を狙っている人）

# 動画構成
動画の中身の内容・構成に関しても、以下で行こうかなと感じております。
1.動画冒頭は自己紹介&実績解説
2.軽く動画のアップロード方法、サムネ、動画のファイル名について解説する
ここで問題提起入れる
「コレは全体の1%でコレだけやっても結果でない」
「伸びる人と伸びない人の違いは、動画をアップロードする"前"に何をしているかです。多くの人は、アップロード方法を覚えた瞬間に、自分の撮りたい動画をなんとなく上げてしまう。
これ、YouTubeで一番やっちゃいけないことです。」

3.でも、多くの方はここで大きなミスをしてしまう。それは適当に自分の好きなような動画をあげてしまうこと
「好きな動画を適当にあげてもダメ。じゃあなぜダメなのか？
ここで多くの人は『再生数が少ないから』『サムネが悪いから』と考えるんですが、
実はもっと根本的な理由があります。
そもそもYouTubeが、あなたの動画を誰にも届けていない。
これ、聞いたことありますか？YouTubeには動画をユーザーに届ける仕組みがあるんです。あなたがラーメン屋を開いたとして、Uber Eatsに登録しますよね。
でも、メニュー名が『料理A』、ジャンルが『未設定』、写真もない。
こんな店に、Uber Eatsはお客さんを届けてくれますか？」

4.大前提のYouTubeのアルゴリズムの解説、狙ってYouTubeを伸ばす方法の解説
「ここまでで、YouTubeの仕組みはわかってもらえたと思います。
YouTubeは『このチャンネルはこのテーマで視聴者を満足させる確率が高い』と判断したチャンネルにインプレッションを配る。
じゃあ、YouTubeにそう認識させるにはどうすればいいのか？」
「ここで出てくるのが、チャンネル設計です。
チャンネル設計って聞くと難しそうに感じるかもしれませんが、
やることはシンプルで、『誰に・何を・どんな独自性で届けるか』を決めるだけです。」

5.YouTubeで伸ばすためにはチャンネルのコンセプトを明確にしないとYouTubeからインプされない的な訴求をする
6.チャンネル設計の話に移る→次にジャンル選定が重要であることを解説
7.差別化ポイントの重要性
8.外注化について解説
9.セールスパート（行動、投資の教育）

の流れで解説をしたいです。
