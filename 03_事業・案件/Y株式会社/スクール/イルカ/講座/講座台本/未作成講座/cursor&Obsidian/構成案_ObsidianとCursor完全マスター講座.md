# 「ObsidianとCursor」完全マスター講座 構成案（全12章）

**テーマ:** ObsidianとCursorを活用し、脳を拡張して「再現性のあるヒット企画・台本」を生み出す
**ターゲット:** YouTube運用中級者〜上級者、AIを使っているが成果が頭打ちの人、思考の整理が苦手な人
**動画本数:** 12本
**想定文字数:** 1動画あたり4,000〜6,000文字（合計 約48,000〜72,000文字）

---

## 全体像
従来の「AIに丸投げ」する量産型運用から脱却し、自分の思考（ナレッジ）をObsidianに蓄積（脳の拡張）し、Cursor（AIエディタ）を最強の手足として使いこなすことで、他者が真似できない「深みのあるコンテンツ」を高効率で生み出す手法を伝授する。

## 各章のプロット（構成指示書）

### 第1章：なぜ今「脳の拡張」が必要なのか？脱・量産型AI運用のすすめ
*   **ターゲット（悩み）:**
    *   AIを使っても似たり寄ったりの動画しか作れない
    *   AIの進化に追いつけず、将来に不安を感じている
    *   「思考力」や「差別化」の重要性はわかるが、具体的にどうすればいいかわからない
*   **参考元動画:** [第1章基礎編](https://utage-system.com/members/iNstBE6G8m0T/course/qIkgGZnN8TW9/lesson/sNQztenz91BJ) / [第4章実践編](https://utage-system.com/members/iNstBE6G8m0T/course/qIkgGZnN8TW9/lesson/N5X3Wsqi1C8j)
*   **動画の流れ（構成案）:**
    1.  **【導入】** YouTube運用のパラダイムシフト。ツールが進化し、誰もが平均的な動画を作れるようになった今、どこで差別化するか？
    2.  **【問題提起】** 「AIへの丸投げ」は思考停止を招く。必要なのはAIを「道具」として使いこなし、自分の「思考」を拡張すること。
    3.  **【解決策】** 「気づく力」「発想力」「言語化力」が資産になる時代。
    4.  **【未来】** ObsidianとCursorを使うことで、自分の脳みそをデジタル空間に再現し、24時間働く「最強の分身」を作るロードマップ提示。

### 第2章：最強の武器「Obsidian」と「Cursor」の導入と役割
*   **ターゲット（悩み）:**
    *   情報の整理が苦手で、過去のアイデアを活かせない
    *   ObsidianやCursorという名前は聞くが、難しそうで手が出せない
    *   従来のフォルダ管理に限界を感じている
*   **参考元動画:** [第2章基礎編](https://utage-system.com/members/iNstBE6G8m0T/course/qIkgGZnN8TW9/lesson/HG3qiFghlJy9) / [第5章実践編](https://utage-system.com/members/iNstBE6G8m0T/course/qIkgGZnN8TW9/lesson/DgQlL8lgNTfx)
*   **動画の流れ（構成案）:**
    1.  **【概念】** フォルダ管理は「情報の墓場」。人間の脳は「リンク（連想）」でできている。
    2.  **【ツール紹介】**
        *   **Obsidian:** 第二の脳。情報をネットワーク状につなぎ、ナレッジを蓄積する場所。
        *   **Cursor:** 最強の手足。Obsidianの情報を読み込み、実行・出力するAIエディタ。
    3.  **【実践】** 実際の画面（グラフビュー）を見せ、脳内が可視化される様子を解説。
    4.  **【マインド】** 「難しそう」はチャンス。面倒くさいことをやるからこそ差別化になる。

### 第3章：視聴者の心を操る「言葉の微差」と「含意」の正体
*   **ターゲット（悩み）:**
    *   サムネイルやタイトルのクリック率が低い
    *   言葉選びのセンスがないと感じている
    *   AIが出すタイトルがいまいち刺さらない
*   **参考元動画:** [第1章基礎編](https://utage-system.com/members/iNstBE6G8m0T/course/qIkgGZnN8TW9/lesson/sNQztenz91BJ) / [第2章基礎編](https://utage-system.com/members/iNstBE6G8m0T/course/qIkgGZnN8TW9/lesson/HG3qiFghlJy9)
*   **動画の流れ（構成案）:**
    1.  **【理論】** クリックするかどうかは「言葉の印象（微差）」で決まる。「閲覧注意」などの強い言葉も使いどころを間違えれば逆効果。
    2.  **【事例】** 「ライオン」＝強い、「カレー」＝人によって好みが違う。言葉には「含意」がある。
    3.  **【実践】** 自分の感覚（エゴ）ではなく、視聴者の「普通」を知る。AIを使って言葉のニュアンス（微差）を大量に出し、最適なものを探る方法。
    4.  **【結論】** 1ミリのズレをコントロールする者がYouTubeを制する。

### 第4章：「普通」を疑え！自チャンネルにおける「成功定義」の策定
*   **ターゲット（悩み）:**
    *   動画の良し悪しを感覚で判断してしまっている
    *   「成功」の基準が曖昧で、改善の方向性が定まらない
    *   他人の成功事例を真似しても結果が出ない
*   **参考元動画:** [第2章基礎編](https://utage-system.com/members/iNstBE6G8m0T/course/qIkgGZnN8TW9/lesson/HG3qiFghlJy9) / [第3章実践編](https://utage-system.com/members/iNstBE6G8m0T/course/qIkgGZnN8TW9/lesson/dWQhffcwI2sz)
*   **動画の流れ（構成案）:**
    1.  **【導入】** 「面白い動画」の定義は全員違う。あなたのチャンネルにとっての「成功」とは？
    2.  **【基準値】** 再生数、維持率、クリック率などの数値で「成功ライン」を明確に定義する。
    3.  **【事例】** ターゲット（例：痩せたい人）によって「美味しいカレー」の定義が変わる話。
    4.  **【ワーク】** 自分のチャンネルの過去データを分析し、AIに学習させるための「基準値」を作成する。

### 第5章：ヒットの法則を解明する「期待」と「裏切り」の構造分析
*   **ターゲット（悩み）:**
    *   企画がマンネリ化している
    *   「意外性」の作り方がわからない
    *   視聴者を飽きさせない構成を作りたい
*   **参考元動画:** [第1章基礎編](https://utage-system.com/members/iNstBE6G8m0T/course/qIkgGZnN8TW9/lesson/sNQztenz91BJ) / [第2章基礎編](https://utage-system.com/members/iNstBE6G8m0T/course/qIkgGZnN8TW9/lesson/HG3qiFghlJy9)

    1.  **【本質】** 全ジャンル共通のヒット構造：サムネで「期待」させ→良い意味で「裏切り」→内容で「回収」する。
    2.  **【分析】** 「良い裏切り」と「悪い裏切り（炎上、期待外れ）」の違い。ジャンル（ニュース、物語、雑学）ごとの裏切りパターンの違い。
    3.  **【AI活用】** AIに「このジャンルにおける良い裏切りパターン」を分析・分類させる手法。
    4.  **【結論】** 構造を言語化し、再現可能な型にする。
