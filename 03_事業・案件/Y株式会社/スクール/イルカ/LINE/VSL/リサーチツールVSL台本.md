## 本編

### チャプター1. この動画は誰に見てほしいか
- 視聴者の潜在的な悩みや疑問を訴求する
- 「YouTubeを頑張っているのに伸びない」「何を投稿すればいいかわからない」「リサーチが大事なのはわかるけど、具体的にどうすれば？」
- 仲間意識を作る：「これ、昔の僕も本当にそうでした」「頑張ってるのに成果が出ない時って、かなりしんどいですよね」
- 「やる気がないんじゃなくて、やり方が整理されていないだけなんです」

### チャプター2. 自己紹介
- 実績の羅列で専門性を見せる
  - YouTube運用の実績
  - 複数ジャンルでの経験
  - 銀盾、月商などの具体数字
- ポンコツエピソードを入れて親近感を作る
  - 昔は毎回YouTubeを開いて、気になる動画をなんとなく保存していた
  - でも見返す頃には何が良かったのかわからなくなる
  - 競合分析のつもりが、ただ動画を見て時間が溶けるだけだった
  - 頑張って作ったのに伸びない
  - その原因が、作業量ではなくリサーチ精度にあると気づいた
- ポンコツエピソードの後、理想の未来を見せる
  - 「リサーチ環境を整えてからYouTubeが本当に変わりました」
  - 「だから今の仕組みが必要だと断言できます」

### チャプター3. なぜ「リサーチ」「リサーチツール」が重要なのか
- YouTubeは `良い動画を作るゲーム` ではなく `見たい動画を先に当てるゲーム`
- つまり、企画の勝率が最重要
- 企画の勝率を上げるのがリサーチ
- 再生されないのは才能がないからではない
- 視聴者が求めているテーマとズレているだけ
- このズレを埋めるのがリサーチ

#### ここで地獄を見せる
- リサーチせずに感覚で投稿し続けると…
  - 頑張って作ってもズレていたら伸びない
  - 伸びないから自信がなくなる
  - 自信がなくなるから投稿が止まる
  - 投稿が止まるから成果も止まる
  - 結果としてYouTubeがしんどくなる

#### ここで天国を見せる
- リサーチ環境が整っていると…
  - 需要があるテーマから逆算できる
  - タイトル、切り口、競合分析が楽になる
  - 投稿の迷いが減る
  - 思いつきではなく、根拠を持って企画が立てられる
  - YouTubeを継続しやすくなる

#### 緊急性と再現性の訴求
- 今この瞬間もリサーチをやっている人とやっていない人で差が開いている
- リサーチは才能ではなく環境。環境を整えれば誰でも再現できる

### チャプター4. 実際どうやるのか（自分じゃできないよねと気づかせる）
- 手作業での一般的なリサーチ方法を軽く見せる
  - 競合チャンネルを見る
  - 伸びている動画を見る
  - タイトルやテーマを比較する
  - 投稿頻度や再生の傾向を見る
  - スプレッドシートにメモする
- 「やり方としては正しいんです。でも…」と気づかせる
  - 毎回これをやるのはかなり大変
  - 手作業だと抜け漏れが出る
  - スプレッドシートの関数エラーに悩まされる
  - 動画の画面とスプシの画面を行き来して時間が溶ける
  - 続かない。結果として、また感覚に戻ってしまう

#### ここで商品の強みから逆算した設計
- 手作業リサーチの「ここが大変」を見せるポイントは、バズミエの強みで解決できる箇所から逆算する
  - 「100万本以上の動画を手作業で見るのは不可能」→ バズミエは一括網羅
  - 「拡散率やコメント率を手計算するのは面倒」→ バズミエは自動表示＋ソート可能
  - 「スプシとYouTubeの画面を行き来する」→ バズミエはツール内にメモ機能がある
  - 「弱小チャンネルの伸びた動画を見つけるのが難しい」→ バズミエは拡散率で発見できる
  - 「今バズっている企画を把握しにくい」→ バズミエはトレンド・急上昇キーワードを自動抽出
  - 「コメントから視聴者の需要を読み取るのが大変」→ バズミエはコメントAI分析で自動抽出
  - 「再生の初速が良い動画を判別しにくい」→ バズミエは再生数推移グラフで一目でわかる

### チャプター5. 商品の強みの解説
- 機能の羅列ではなく、悩みとの対応で解説する
- 「この悩み → バズミエだとこう解決できます」の形

#### 悩み → 解決の対応
- 何をリサーチすればいいか迷う → ジャンル・長尺/ショートで絞り込みできる
- 競合分析に時間がかかる → 100万本以上の動画データを一括で網羅
- 伸びる企画のヒントが見えない → 拡散率ソートで弱小チャンネルの伸びた動画も発見
- 再生の初速が良い動画がわからない → 再生数推移グラフで一目で判別
- コメントから需要が読み取れない → コメントAI分析で視聴者の悩みを自動抽出
- スプシとの画面行き来が面倒 → ツール内メモ機能でリサーチが全て完結
- 台本やコメントをまとめたい → CSVでダウンロード可能
- 今バズっている企画を知りたい → 急上昇キーワード・トレンドを自動抽出
- 競合の収益が気になる → チャンネル・動画ごとの想定収益を自動算出
- AIに企画の壁打ちをしたい → リサーチツール内でAI壁打ち可能

#### 価格の見せ方
- 月額4,980円は最後の方で伝える
- 先に `時間短縮` `無駄な動画を減らせる` `企画精度を上げられる` を伝える
- その上で `動画1本外す損失` より安いと認識させる

### チャプター6. CTA
- 売り込み感ではなく、自然に `試してみようかな` を作る
- 7日間無料トライアルの案内
- 定員に達し次第終了する旨を伝える
- 下のリンクから確認してほしいと案内する
