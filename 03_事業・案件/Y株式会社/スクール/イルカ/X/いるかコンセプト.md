## 2. ペルソナ

### 2.1 ①35歳サラリーマン

- 情報
    - 性別：男性
    - 年齢：35歳
    - 職業：都内の中堅企業に勤めるサラリーマン
    - 家族：妻と5歳の子供が1人
- 現状の不満
    - 給料は上がらないのに、物価や税金は上がる一方で、将来が漠然と不安
    - 毎日朝から晩まで真面目に働いているが、このままでいいのか疑問を感じている
    - 本業が忙しく、平日に自由な時間はほとんどない（帰宅後、寝るまでの1〜2時間程度）
    - 会社に副業がバレるのは絶対に避けたいので、顔出しは絶対にNG
- 願望・欲求
    - まずは「月10万円」の副収入を得て、生活の不安をなくしたい
    - 子供の教育費や、たまの家族旅行に気兼ねなくお金を使いたい
    - 最終的には「会社の給料と同等（月30〜40万）」を稼げるようになり、「会社を辞める」という選択肢を手に入れたい
    - 「自分には特別なスキルはない」
- イルカアカウントを見る理由
    - YouTubeが稼げるらしいとは知っていたが「顔出し」や「編集スキル」が壁になり、自分には無理だと諦めていた
        - 「AIで」「顔出し無しで」できるなら自分にも唯一やれるチャンスかもしれない、と希望を感じる
    - 「外注化」という言葉に、「自分がずっと働き続けなくても収入が生まれる仕組みが作れるのでは？」と最終的な理想像（時間の自由）を垣間見る
- 外注化に対する不安
    - 外注化できる自信がない。自分が依頼側に立って良いかわからない
    - 自分が指示できるか自信がない
    - そもそもどうやって外注して良いのか？どこのサイトで外注して良いのか？
    - どのような金額で外注するのかわからない
- 信頼できないコンテンツへの対策がわからない
    - ✅動画の構成が毎回ほぼ同じ
    - ✅オープニング・エンディングが固定
    - ✅サムネイルの「型」が毎回同じ
    - ✅概要欄が毎回コピペ
    - ✅投稿本数が異常に多い（毎日投稿など）
    - ✅運営者の人格が見えない（機械音声のみ）
    - ✅ネットの情報をまとめただけの内容
    - ✅編集が簡易的で誰でも作れそう
- AI活用における3大不満
    - 1つ目：AIを使っているのに、全然楽になっていない
        - 量産しているけど、伸びない
        - 作業感が強くなって、正直しんどい
    - 2つ目：どこを直せばいいのか分からない
        - 台本なのか、企画なのか、構成なのか
        - 判断基準がなくて、改善ができない
    - 3つ目：このまま続けていて大丈夫なのか不安
        - 最近のBANや収益停止の話も含めて、「自分もいつか来るんじゃないか」という怖さがある
- AIを入れて逆に苦しくなっている人が多い（最大の不満）
    - 作業は増えている
    - 量産はしている
    - でも伸びない
    - 正直、何のためにやっているのか分からない
    - 「AI工場」みたいになっている

### 2.2 ターゲット具体化（コミュニティ設計用）

- Q1. ターゲットはたった一人まで具体化できていますか？（年齢・職業・悩み・願望・行動パターン・感情）
    - 30代後半で職業は会社員（比較的安定給の職業）
    - お金がなく子供の学費が払えない
    - お金が理由で理想の生活ができない & 行動が止まっている & ストレスがかかっている
    - 行動する気力はあるが何をして良いかがわからない
    - 過去に他の副業に挑戦したが、教材を買っただけで行動できず3ヶ月で挫折した
- Q2. その人が「人生が変わった」と感じる瞬間は、具体的にどんな状態ですか？
    - お金のストレスがなくなった状態
    - お金の不安がなくプライベートを過ごすことができている状態
    - 自分の人生はいつでも自分で変えられる成功体験を積み上げた先に、自分の人生をコントロール・制御している感覚がある状態を得た時に、人生変わったと感じる
- Q3. その人が今まで行動できなかった理由（障壁）は何ですか？
    - SNS時代、情報量が多くそもそもどこから手をつけて良いかわからない & やり方がわからない
    - どの発信者を見て良いかわからない
    - 誰かと一緒でないと何かに取り組むことができない
    - 努力していても自分がやっていることが正しいか不安で手につかない
    - 実績があり信頼できる誰かに明確な道標を提示してほしい
    - 無料＝価値のない情報だと決めつけている（重要度が低い状態）ため、お金を払った有料コンテンツが見たい

### 2.3 ターゲットの深層悩み

- **1. AIに飲まれている**
    - 効率化のためにAIを入れたはずが、毎日ゴミのような動画を量産するだけの「工場作業」になってしまった。何のためにYouTubeを始めたのか分からない
    - 数字（再生数）は出ても、自分のスキルが上がっている実感が全くない。また数字がなかなか上がりきらない
    - ただツールに使われているだけで、虚しい
- **2. スキル不足への不安**
    - 直近AIツールありきで始めた人は、そもそもAI前提のYouTube市場で戦い、AI前提の台本執筆・企画選定の流れが染み付いているため、本物志向の人がどのように企画立案・台本執筆しているか？わからないし、手法が知りたい
    - AI前から参入している人は、AIの波に飲まれてAIを使う方にシフト→ただ、AI前提の企画立案・台本執筆の流れが染み付いてしまい、なかなか元の状態に戻れない
    - AIを手放すことで、周りと差が生まれるのでは？という不安がある
    - 周囲の人が発信やnote、brainでAI系のコンテンツ（小手先的なもの）を出すため、つい買ってしまう。勉強にはなるが、自分自身の真の成長にはなる。ただ、なかなか収益が上がらない
- **3. 「本物」と自分との差に対する焦り**
    - 「なぜ、あの人は（みこさわ）、AIを使っているのに、ファンもついていて、楽しそうに稼ぎ続けているのか？自分と何が違うのか？」
    - AIの波に乗っているはずなのに、むしろAIに飲み込まれて、他者との差別化ができなくなってしまった

### 2.4 自分が成功している要因

- 流行りのAIは使いつつも、ベースは地道なリサーチと企画力を鍛えることを継続して行ったこと
- ただの量産でお金のためにやるのではなく、真剣にYouTubeと向き合っているから楽しいしやりがいがある
- 量産系の人たちはお金を常に追うため、収益が出ず疲弊、ライン工のような生産体制を敷いているため、面白みがなく単調作業になっている
- 本気で向き合うからこそ、本当にYouTubeを楽しく運営できる
- 楽しいから継続して行うことができる
- 結果として、収益が上がる仕組みを作ることができた
- 結局は自分自身の実力を上げると同時にAIを活用することで、累乗に収益とスキルが伸びるし、1000万円以上スケールさせていくためには、AIの知識ではなくワーカーさん（人材）を成長させなければいけない

## 5. コンセプト

### 5.1 メディア型のX

#### プレキャンでの差別化

- とにかく質を高くする
    - わかりやすさ
        - 小学生でもわかる文
            - 簡単な単語
            - 言い切り
            - 一文80文字以下
        - ユーザーの目線に立つ
            - ユーザー理解
                - ペルソナ設定
                - バズ投稿のリサーチ
    - ボリューム
        - 最後まで見させることも重要
            - PASONA法
        - 8分以上の動画を20本
        - 動画だけでなく、テキストでも情報を追加
            - ポストと動画を接続させる
    - 内容の濃さ
        - 欲しい情報を提供する
            - ポストと内容がズレすぎてない∧役に立つ本質情報
                - ポストで発生した欲求はテキストでちゃんと満たしつつ、役に立つ情報を動画で提供
        - 目新しさ
        - 役に立つ

#### マスに刺す

- インプを高める
    - **3つのNOT**
        - NOT READ → 強烈なフックを用意する
        - NOT BELIEVE → 権威性・証拠で突破
        - NOT ACT → 今すぐ行動する理由を示す
    - **「上手い文章≠売れる文章」**
        - 差は「情報を伝えること」と「感情を動かすこと」どちらを意識してるか
            - 上手い文章
                - 文法的に正しい
                - 美しい表現が使われている
                - 論理的で、情報が整理されている
                - 納得感がある
                - だが、読み終わった後、特に何も行動しない
            - 売れる文章
                - 読み手の感情を動かす
                - 自分のことだと思わせる
                - 読んだ後、何かしたくなる衝動が生まれる
    - **「So What？テスト」**
        - ペルソナ設定の重要性
            - みんなに向けた文章は、誰の心にも刺さらない
                - 明確なペルソナ設定が重要
        - 手順
            1. 特徴を書き出す
            2. 「So What？（だから何？）」と自問する
            3. 答えが出たら、さらに「So What？」を繰り返す
            4. 顧客の感情が動く答えにたどり着くまで続ける
        - 具体例：プロテインの場合
            - 特徴：「タンパク質含有量が25g」
            - So What？：「筋肉の材料が効率的に摂れる」
            - So What？：「筋トレの効果が出やすくなる」
            - So What？：「鏡を見るたびに『お、ちょっと変わってきた？』と実感できる」
            - So What？：「自信がついて、夏の海やプールが楽しみになる」
    - **PASONA法**
        - 構成要素
            - **Problem（問題提起）**
                - これは自分の事だと思わせる
                    - 質問型：「最近、鏡を見るのが嫌になっていませんか？」
                    - 共感型：「会議で発言しようとしても、声が震えて、結局何も言えずに終わる。」
                    - 衝撃型：「実は、40代男性の73%が、このままでは老後資金が足りません。」
                - 具体的なシーンを描く
                - 五感を使う
            - **Agitation（煽り・共感）**
                - 単に「煽る」だけではなく読み手の感情に寄り添い、「この人は私の気持ちをわかってくれる」と思わせる
                    - ①現状の痛みを深掘り：「気づけば同期は次々と昇進し、自分だけが取り残される」
                    - ②感情への共感：「努力しているのに報われない。その悔しさ、痛いほどわかります」
                    - ③希望の光を見せる：「実は、同じ悩みを抱えていた人が、今では...」
            - **Solution（解決策）**
                - 単に解決策を提示するのではなく、「救世主」感を出す
                    - ①期待を高める前フリ
                    - ②解決策の提示
                    - ③解決できるメカニズム
                    - ④客観的証拠
                    - ⑤ベネフィットの再確認
            - **Narrow down（絞り込み）**
                - 「いつか」「今度」「もう少し考えてから」この心理を打破する。要はNOT ACTの打破
                    - 数量限定
                    - 期間限定
                    - 特典限定
                    - 条件限定
                - 理由の明確化・損失回避バイアスの利用・希少性の可視化（具体的な数値）
                    - 例）品質担保のため〜 / 次回入荷は3か月後〜 / 残り13個
            - **Action（行動喚起）**
                - 読み手に具体的なアクションを促す
                    - ①具体的な指示：申し込んでください、じゃダメ。下の緑のボタンを押して申し込んでください。くらいの具体性
                    - ②行動のハードルを下げる：クレカ不要・申し込みはたった30秒
                    - ③行動後の流れを提示：「お申し込み後、すぐに確認メールが届きます。その後、24時間以内に商品を発送いたします」
                    - ④最後の一押し：迷っているなら、まずは無料体験だけでも
        - PASONAの黄金比率
            - Problem（問題提起）：10%
            - Agitation（煽り・共感）：30%
            - Solution（解決策）：30%
            - Narrow down（絞り込み）：20%
            - Action（行動喚起）：10%
            - 重要なのは、SolutionまでにPASONAの40%を使うこと → すぐに商品説明に入らない
    - **ストーリーテリング**
        - 単なる事実としてではなく、物語として情報を伝える
            - データで示された情報の12倍以上記憶に残る
        - 3つの型
            - ビフォーアフター型：問題を抱えた状態から、解決した状態への変化を描く
            - 共感ストーリー型：読み手と同じ悩みを持つ人物の物語を通じて、「自分と同じだ」と思わせる
            - 発見・気づき型：新しい視点や解決策との出会いを中心に据えた型
                - 例：「『投資なんて、金持ちがするもの』そう思っていた私が、あるカフェで隣の席の会話を偶然聞いてしまった。『月3万円の積立で、20年後には...』普通のOLらしき女性の話に、私は耳を疑った。」
    - **フック**
        - 8つの型
            1. 質問型：読み手に「YES/NO」を考えさせる → 例：「月10万円の副収入、興味ありませんか？」
            2. 数字型：具体的な数字で信憑性と興味を引く → 例：「年収300万→1000万円を実現した28歳の秘密」
            3. 逆説型：いわゆる常識の破壊 → 例：「痩せたいなら、食べろ」
            4. 恐怖・不安型：失う恐怖や不安を刺激し、行動を促す → 例：「知らないと損する、2025年の税制改正」
            5. 好奇心型：「それって何？」と思わせる → 例：「Google社員だけが知っている検索の裏技」
            6. ベネフィット型：得られる利益を直接的に示す → 例：「初心者でも月50万円稼げるビジネスモデル」
            7. ニュース型：新しい情報として提示する → 例：「【2025年最新】AIで変わる仕事の未来」（らっこ講座でいう新規性）
            8. 共感型：読み手の状況や感情に寄り添う → 例：「『また今月も貯金できなかった...』そんなあなたへ」
        - 黄金の方程式
            - ターゲット + 数字 + ベネフィット + 簡単さ
                - 例：「40代のお父さんが、1日5分で3ヶ月10kg痩せた方法」
        - その他のテクニック
            - ツァイガルニク効果：未完成のものに注意が向く心理を利用 → 例：「実は、あなたの●●が原因でした」
            - バンドワゴン効果：みんながやっていることに興味を持つ心理 → 例：「今、ビジネスエリートが注目する」
            - 希少性の原理：限定感で行動を促す
            - 権威への服従：権威ある人物や組織の力を借りる
    - **CTA**
        - 7つの要素
            1. 動詞で始める：×「申し込みはこちら」→ ○「今すぐ申し込む」
            2. 具体的である：×「もっと見る」→ ○「PDFを今すぐダウンロード」
            3. 緊急性を演出する
            4. ベネフィットを含める → 例：「5分で保険料を削減する」
            5. リスクを取り除く → 「30日間返金保証」
            6. 簡単さを強調する → 「たった30秒で登録完了」
            7. 視覚的に目立たせる
        - その他のテクニック
            - 損失回避バイアス：「今すぐ始める」より「機会を逃さない」
            - 社会的証明：「10,234人が既に始めています」
            - 選択のパラドックスを避ける：選択肢は最大3つまでに絞る
    - **PREP法**
        - 感情に働きかけるPASONA法とは対照的に、論理的で明快な文章構成法
            - この2つを使い分けることが大事
        - 構成要素
            - Point（結論）：最初に結論を述べる
            - Reason（理由）：なぜそう言えるのか理由を説明
            - Example（例）：具体例で理解を深める
            - Point（結論）：最後にもう一度結論を強調
        - 何故効果的なのか
            - 初頭効果：人は最初に聞いた情報を最も強く記憶する。結論を先に述べることで、最も伝えたいメッセージを確実に印象づけられる
            - 認知負荷の軽減：「これから何の話をするのか」が最初に分かることで、読み手の脳は情報を整理しやすくなる
            - 確証バイアスの活用：最初に「朝活は人生を変えます」と結論を述べると、朝活はいいんだな、という前提で読み進めてもらえる
    - **ベネフィット訴求**
        - So What？テスト
            1. 特徴を書き出す
            2. 「So What？（だから何？）」と自問する
            3. 答えが出たら、さらに「So What？」を繰り返す
            4. 顧客の感情が動く答えにたどり着くまで続ける
        - 3つのレベル
            - レベル1：機能的ベネフィット ─ 商品の機能から直接得られる価値
                - 例「時短になる」「便利になる」「安くなる」
            - レベル2：情緒的ベネフィット ─ その結果感じる感情的な価値
                - 例「安心できる」「楽しくなる」「自信が持てる」
            - レベル3：自己実現的ベネフィット ─ なりたい自分になれるという最高次の価値
                - 例「理想の自分になれる」「人から認められる」「夢が叶う」
        - 偽ベネフィットに注意
            - 「高品質をお届けします」「満足度95%」「最新技術を採用」
            - 要はユーザーが価値を感じている瞬間の情景をイメージできるか
    - **返報性の原理**
        - プレキャンの中にリポスト呼びかける内容いれてもいいかも
    - **希少性**
        - 具体的な数字
        - 理由
