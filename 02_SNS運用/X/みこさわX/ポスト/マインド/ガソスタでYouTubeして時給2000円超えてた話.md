# ガソリンスタンドでYouTubeして時給2000円超えてた話

ぶっちゃけ・・・
大学生の時、ガソリンスタンドのバイト中にYouTubeで稼いでた。

監視業務だったから、暇な時間にPC持ち込んで作業してた。

バイトの時給＋YouTubeの収益
合算したら時給2000円は余裕で超えてた。

顔出しなし。場所も関係ない。
パソコン1台あればどこでもできる。

バイト先でも、通勤中でも、家のソファでも。
これがYouTubeの最大のメリット。

当時の私は月7〜9万のバイト代に不満だらけだった。
時給は上がらない。
シフトも選べない。
会社側に主導権を握られてる感覚がずっとあった。

だから「この状況を自分で変えるしかない」と思って副業を始めた。

バイト中にビジネス書を200冊以上読んで気づいたのは
「本を読むだけでは生活は1ミリも変わらない」ということ。

行動して、自分で稼ぐ経験を積まないと何も変わらない。

今バイトや会社で暇な時間がある人。
まずはその時間にYouTubeの音声学習からでも初めてみてください。
移動中でも、作業中でも、それだけでいい。

その小さな一歩が、自分で稼ぐ力の土台になります。
