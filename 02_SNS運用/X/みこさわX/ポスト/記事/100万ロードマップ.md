## なぜ今、YouTubeなのか？

### マーケティングスキル ＝ 人生のスキル

「YouTubeで稼ぐ」と聞くと、多くの人は動画編集や再生数のテクニックを想像する。

でも、私はそうは思わない。

**YouTubeで身につくスキルの本質は「マーケティング」であり、マーケティングの本質は「ブランディング」であり、ブランディングの本質は「信頼構築」だ。**

信頼を構築する力。

これは、人生のありとあらゆる場面で活きる。

- 就職活動で、面接官に「この人と働きたい」と思わせる力
- 恋愛で、相手に「この人といたい」と感じさせる力
- 友人関係で、「この人は信頼できる」と思われる力
- 営業で、「この人から買いたい」と決断させる力
- チームマネジメントで、「この人についていきたい」と思わせる力

YouTubeを本気で運営すると、これら全てのスキルが同時に鍛えられる。

なぜなら、YouTubeは「画面の向こうにいる見ず知らずの人に、たった数分で信頼してもらう」という、最も難易度の高い信頼構築ゲームだからだ。

ここで磨いたスキルは、対面の仕事や日常の人間関係では、もっと楽に使える。

私はYouTubeを「お金を稼ぐツール」ではなく、**「人生を攻略するための最強の訓練場」**だと思っている。

### YouTube市場は、今も右肩上がりで伸び続けている

「YouTubeってもうオワコンじゃないの？」

こういう声を、いまだに聞く。

結論から言うと、全くの逆だ。

2025年7〜9月期、Googleの親会社アルファベットの決算では、**売上高・最終利益ともに四半期ベースで過去最高を記録**した。増収増益は10四半期連続。YouTubeの利用も堅調で、AIサービスとの連携による成長が続いている。

さらにGoogleは、2025年の設備投資額を従来の850億ドルから**910億〜930億ドルに増額**。AI向けデータセンターの整備を中心に、プラットフォームへの投資を加速させている。

参考：[米グーグル決算は売上高・最終利益が過去最高…好調な生成ＡＩサービス需要、ユーチューブ利用も堅調（読売新聞）](https://www.yomiuri.co.jp/economy/20251030-OYT1T50032/)

2027年、2028年まではこの成長トレンドが続くと私は見ている。

世界最大のプラットフォームが、過去最高の投資をして、まだ伸び続けている。

**この時代の波に乗らないのは、正直もったいない。**

今からYouTubeを始める人は「遅い」のではなく、**「市場が成熟しきる前の、最後のチャンス」**にいる。

## STEP1：ジャンル選定

### 選定基準①：需要の大きさ

ジャンル選びで最も重要な要素。
どれだけいい動画を作っても、見たい人がいなければ再生数は一生伸びない。
YouTubeに限らず、全てのSNSで共通する超重要な原則。

**需要の重要性**
- 魚が全くいない場所で高性能な釣竿を使っても魚は釣れない
- 多くの初心者は、これと全く同じことをしてしまっている

**典型的な間違い**
- 「自分の好きなこと」や「作りやすいもの」という理由だけで需要のないジャンルを選ぶ
- 景色がいいからという理由で魚がいない場所を選んでしまう

→ まずは「どんな場所（ジャンル）に多くの人が集まっているか」を徹底的に調査する

### 選定基準②：競合の弱さ

どんなにレッドオーシャンなジャンルに見えても、競合チャンネルの「弱点」は必ずある。
その弱点を改良し、「競合チャンネルに勝てそうかどうか」も大切な判断基準の1つ。

**競合の具体的弱点例**
- サムネの完成度が低い（文字が小さい・配色が悪い・視認性が低い）
- 冒頭30秒に結論やフックがない
- 編集が単調で抑揚がない
- コメント欄に「分かりにくい」「情報が古い」などの不満がある
- 投稿頻度が低い（週1以下）

**弱点探しのコツ**
- 「弱点を探す」という気持ちでYouTubeを見てみると案外面白い
- 上記以外にも弱点の種類は無数にある
- 気になるジャンルがあれば、まずは探してみることを推奨

### 選定基準③：収益性の高さ

どんなに再生されても、収益が低ければ意味がない。
極端な話、1000万回再生されようと1000円しか稼げなかったら嫌。

**YouTubeの収益決定要因**
「広告の単価」と「広告がどれだけ流れるか」の2つで決まる。

**広告単価について**
- ジャンルによって大きく異なる
- 再生単価が0.1円以下のジャンルもあれば、2円以上のジャンルもある
- 「YouTubeの再生単価は0.1円」という噂が流れているが、実際はジャンルによって全く異なる

**イルカの実例**
- ある動画の収益は1000再生あたり1600円 → 再生単価1.6円

**再生単価の違いによる収益差**

| 再生単価 | 10万回再生の収益 |
|---|---|
| 0.1円 | 1万円 |
| 2円 | 20万円 |

→ 同じ再生数でも、ジャンルが違うだけで**20倍**の差が出る

**一般的に広告単価が高いジャンル**
- ビジネス系・金融系・健康系・教養系
- 簡単に言うと「お金・健康・学び」に関するジャンルは単価が上がりやすい

**広告が流れる回数について**
- 動画の長さによって決まる
- 8分以上の長い動画は、動画内に広告を複数回入れられるため収益が増えやすい
- 単価だけでなく、そのジャンルの動画がどれくらいの長さのものが多いのかも調べておく

### 2025年注目3ジャンル

1. **日本称賛系**（海外反応 × 日本文化・技術・歴史）
2. **シニア向け恋愛解説**（中高年の恋愛・人間関係悩み）
3. **2chまとめショート**（掲示板を会話形式でまとめ）

## STEP2：競合リサーチ

> 「リサーチを制す者がYouTubeを制す」

めちゃくちゃめんどくさいからこそ差がつく。
目的は、ライバルの成功パターンを戦略的に盗むこと。

### 分析対象の選定

- 人気チャンネル10個 × 再生上位3本 ＝ **計30本**
- 30本の理由：2〜3本では「たまたま」を排除できない
- ここで踏ん張れば後々楽になる

### 5つの分析ポイント

**① サムネイル分析**
- 色の傾向（赤×黄・青×白・黒背景）
- 文字（スマホで読めるか・画面の占有率）
- 頻出ワード（知らないと損・○選・衝撃）
- 人物の表情（驚き顔・指差し）

**② タイトル分析**

3つの要素に分解して考える。

| 要素 | 役割 | 例 |
|---|---|---|
| メイン | 動画主題 | 筋トレ・YouTube始め方 |
| サブ | 補足情報 | 初心者向け・家でできる |
| インパクト | 感情トリガー | 知らないと損・たった5分 |

例：ダイエット ＝ メイン「ダイエット」+ サブ「全身脂肪燃焼」+ インパクト「地獄の」

**③ 冒頭30秒の構成（4パターン）**
- **結論先出し型**：「結論から言うと〜」
- **エピソード導入型**：「先日こんなことが〜」
- **問題提起型**：「あなたは○○で悩んでませんか？」
- **数字インパクト型**：「月100万円稼げました」

**④ 投稿頻度・タイミング**
- 頻度：毎日・週3・週1のペース
- 時間帯：平日夜・土日昼・平日朝
- 継続性：現在も定期投稿中か

**⑤ コメント欄（イルカが実は最も重視している要素）**

コメント欄は視聴者の感情が集まる場所。

- **共感コメント**：「もっと早く知りたかった」「めちゃくちゃ助かる」など → 視聴者に響いている内容や表現を見つけられる
- **不満コメント**：分かりにくい・情報が古い・冗長・信頼性が低い → 改善点を見つけられる。ここを解決できれば競合より優位に立てる

## STEP3：チャンネル設計

差別化ができていなかったり、一貫性が欠如していると詰む。
「このチャンネルだから見たい！」と思わせることが全て。

### ① ターゲットの具体化

「特定の誰か一人に届ける」意識が必要。
一人向けが結果的により多くの人に響く。
「30代男性向け」では不十分。

**5項目を明確にする**

| 項目 | 例 |
|---|---|
| ① 年齢・性別 | 35歳男性 |
| ② 仕事・家族構成 | サラリーマン・家族あり |
| ③ 今の悩み | 残業が多く時間がない |
| ④ なぜYouTube視聴するか | 将来に不安があるから |
| ⑤ 望む未来 | 副収入を得たい |

→ ここを決めることで、作るべき動画・響く言葉・表現方法が明確になる

### ② 唯一無二の価値設計

「なぜあなたのチャンネルを登録するのか？」の理由を作る。

**4要素の掛け合わせ**

| 要素 | 内容 | 例 |
|---|---|---|
| ジャンル | テーマ絞り込み | 筋トレ → 宅トレ → 30代男性向けモテボディ |
| ターゲット | 上記で具体化した視聴者 | ─ |
| 届け方・切り口 | 表現形式 | ドキュメンタリー風・インタビュー形式・ランキング形式 |
| キャラクター・視点 | 話者の立ち位置 | 等身大の先輩・プロの専門家・優しい相談相手 |

**具体例：プログラミングチャンネル**

一口にプログラミングのチャンネルと言っても、切り口次第で色々な見せ方ができる。

- 現役エンジニアの最新技術解説
- 文系大学生の独学アプリ開発記録
- 元お笑い芸人の挫折あるある解説

## STEP4：企画・サムネ作成

> 「クリックされなければ存在しないのと同じ」

動画の伸びは企画とサムネで決まると言っても過言ではない。
ここを疎かにすると永遠に再生数1桁の可能性さえある。

### 企画作成方法

**【リサーチ編】ヒット企画を探す**

1. **新規かつ伸びているチャンネルを参考にする**
   - 立ち上げ半年〜1年で急伸 ＝ 再現性が高い
2. **拡散率で考える**
   - 再生回数 ÷ 登録者数
   - これが3倍以上だといい
   - 例：登録者5万 → 再生15万超 ＝ 企画ヒット
3. **市場平均の1.5〜2倍を参考にする**
   - そのジャンルの平均が10万回なら、15〜20万回の動画を分析

**【立案編】切り口の掛け合わせで無限に企画を作れる**

- 例①「緊急性」×「恐怖」
  → 「【知らないとやばい】プログラマーが本当に使っているサイト3選」
- 例②「ストーリー」×「共感」
  → 「文系・未経験だった僕が、半年でプログラミングで食えるようになるまでにやったこと」

### サムネイル作成

最も意識すべきは**「視認性」**。
一目で内容が判断できなければクリックされない。

**4つの黄金法則**

| 法則 | 内容 |
|---|---|
| ① 3秒で内容が分かる | 視聴者がサムネを見る平均時間は3秒。スマホで文字がはっきり読めることが重要 |
| ② 感情に訴える | 驚き・不安・期待などの感情を煽る。例：「便利なアプリ5選」→「知らないと損する神アプリ5選」 |
| ③ 文字は大きく・少なく | 15文字以内・強コントラスト（黒背景×黄色文字） |
| ④ Z型視線誘導 | 左上→右上→左下→右下の視線移動を意識。最重要情報は左上に配置 |

**実践**
- A/Bテスト前提でサムネ2〜3案作成
- CTR・初動1時間のクリック率と維持率で判断
- タイトルとサムネの一貫性確保（釣りタイトルNG）

## STEP5：台本作成

> 「実はここが収益に直結する」

最も見落とされがちな要素（目に見えないから）。

**視聴維持率 ＝ 収益に直接結びつく最重要要素**
- 同じ再生数でも、維持率の違いで収益は3〜4倍変化する
- 長時間視聴される → 広告が多く表示される → 単価上昇
- 例：維持率低 → 月収20万円 ／ 維持率高 → 月収80万円
- 年単位だと800万円ほど差が出ることも

### 視聴者を最後まで惹きつける台本の3要素

**① 具体：すぐ実践できるノウハウ**

抽象的な話ではなく、明日から使える再現性の高い情報を入れる。

| | 例 |
|---|---|
| 悪い例 | 「コミュニケーションが大切なんです」 |
| 良い例 | 「5分間に名前を3回呼んでください」「『そうですね』で共感 →『ちなみに』で展開してください」 |

→ 「すぐ使えそう」という感情が視聴維持率に貢献する

**② 哲学：独自の思想・価値観**

思想に共感する → ファンになる → 熱心に視聴・共有 → 長期的にチャンネルを成長させてくれる。

例：YouTubeで稼ぐ方法 ＋ なぜYouTubeが人生を変える力を持つのか

**③ ストーリー：感情を動かす体験談（最重要）**

リアルな体験談で「自分ごと」として深い共感を生む。

| | 英会話教材の例 |
|---|---|
| 悪い例 | 「3000語・AI発音・24時間対応」 |
| 良い例 | 「NYのカフェで店員と笑顔で会話できた感動は一生忘れられない」 |

→ 情報を記憶に刻み込み、行動へ導く

### AI活用時の3つの注意点

**① AIは効率化ツールと認識する**
- 出力されたものをそのまま使っていては一生稼げない
- リサーチ・構成案などの「高速化・効率化」は可能
- AIが生み出したものを改善し続ける作業が必須

**② 浅い質問はNG**
- 具体的かつ視聴者理解に基づくプロンプトが必要

| | 例 |
|---|---|
| 悪い例 | 「台本作って」 |
| 良い例 | 「○○の悩みを持つ30代男性が共感する具体エピソードを盛り込んだ台本を」 |

**③ 最重要は視聴者理解**

AIはあくまでツール。視聴者を理解するのは人間の仕事。

## STEP6：外注化

> 「この考えを持てないと稼げない」

あなたの目的は、編集スキルの習得ではなく、時間とお金の自由を手に入れること。

- 動画編集・サムネ作成等は他の人に任せる
- あなたは企画・台本等「頭を使う部分」に集中する

### 失敗しない外注者を選定する4基準

| 基準 | チェックポイント |
|---|---|
| ① 募集文の理解度 | 質問にちゃんと回答しているか。条件を順守できているか（例：1本3000円で募集しているのに5000円で条件を出してくるなど） |
| ② 当たり前なことだけをアピールしていないか | 「期日守ります・真面目です」のみはNG |
| ③ 守秘義務・著作権意識があるか | 他社の制作物を無断公開している人は後々トラブルになる可能性大 |
| ④ コミュニケーション能力 | 素直さ・報連相・基本マナーなどがあるか |

### 運用の3つのコツ

1. **小さく始める** — まずは編集のみ → 次にサムネのみと段階的に外注する
2. **複数人でテストする** — 2〜3人に同じ作業を依頼してクオリティを比較する
3. **長期関係を構築する** — 良い外注者とは継続的な関係を築く

### 投資配分の考え方

- 高い機材を購入するより外注費を優先
- 時間を買って、より価値の高い作業に使う ＝ YouTube成功への最短ルート
- **月10万の外注費 → 月100万収益の構造を作る**
