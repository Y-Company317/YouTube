# YouTubeリサーチ5つの型

⚠️知らないだけで差がつく
完全初心者でもすぐ実践できるYouTubeリサーチ5つの方法

━━━━━━━━━━━
１、ネット検索でジャンルを洗い出す
━━━━━━━━━━━

「非属人 YouTube ジャンル」などで
YouTubeとGoogleの両方で検索してみる

・最低10個のジャンル候補を洗い出す
・1つだけでは比較できない
・複数比較して「勝ちやすさ」や「収益性」の差が見える

ここで「2chまとめ」「ゆっくり解説」「日本称賛系」など
知らなかったジャンルの情報を収集することができます。

━━━━━━━━━━━
２、YouTubeでワード検索する
━━━━━━━━━━━

「解説」のような抽象ワードではなく、具体的な固有名詞で検索する。

・✕「歴史 解説」→ 競合が多すぎて迷う
・◯「江戸時代 解説」→ 伸びてる動画の型が見える

具体にするだけで、そのジャンルでどんな動画が伸びているか一発でわかる。

━━━━━━━━━━━
３、サジェスト検索を辿る
━━━━━━━━━━━

YouTubeの検索窓にキーワードを入れると、自動で候補が出てきます。
これはYouTubeが「視聴者の需要」を元に表示しているもの。

・「解説」と入れる→「ゆっくり」「ゲーム」「歴史」が出る
・そのワードをさらに辿っていく

これを繰り返すだけで、自分が全く知らなかったジャンルが次々と見つかります。

━━━━━━━━━━━
４、人気ワードで横展開する
━━━━━━━━━━━

サジェストを繰り返すと、よく出てくるワードが必ずあります。

・稼ぎ方
・政治
・切り抜き
・〇選
・ビフォーアフター
・完全解説

これらは多くの視聴者が検索している人気ワード。
このワードで調べると、伸びている「動画の型」や「企画の切り口」が見つかります。

━━━━━━━━━━━
５、関連動画を数珠つなぎにする
━━━━━━━━━━━

リサーチ専用のアカウントを作成して、
リサーチするジャンルの動画に表示される「関連動画」を辿っていきます

・リサーチ専用のブランドアカウントを作る
・ベンチマークchの人気動画を最後まで再生
・表示される関連動画を5本ほど連続視聴

5本ほど辿っていくと、全く知らなかったchにたどり着く。
それはアルゴリズムが「この視聴者が次に見たい動画」としておすすめしているものになります。

━━━━━━━━━━━

この5つは、スマホ1つでどれも移動中、お風呂でもすぐにできます。

まずは今日、YouTubeの検索窓を開いて1つキーワードで検索してみるだけでも行なってみましょう。
