# 滑っている企画と伸びている企画の差
選挙制度、政策制度　、〜の裏側、〜の闇は善悪で判断するため、視聴者が疲れる、学びにならないが、伸びている企画である〜史、〜完全解説、は学び、知的要素がある、学び要素がある→安心感に繋がるのでしょうか？
また、チャンネルコンセプト的に、フロントでは大々的に知的好奇心や安心感、完全網羅感の訴求をして、中身ではコンセプトのイジリ要素があるため、落とし要素に偏りすぎないけど、程よく落としがある、視聴者意見の代弁によるスカッと感がある様子がはまるのでしょうか？理由を深掘りたい。


# コンセプトと企画
また、企画として、チャンネルコンセプトと合致しているイジリがいがある企画や人物、対象である「鳩山さん」「高市さん」「中道連合」を企画することで伸びている。（チャンネルコンセプトと企画がかけ合わさると企画力が上がる）

# コンセプトと視聴者の需要
コンセプトである「一番面白い解説」は、面白い & 分かりやすい解説はその事象をより深い段階で知っていなければ分かりやすく解説できない → これらのため、視聴者にはこの人の動画を見ることで「博識になれる」「楽しみながら」「時間を無駄にしない」などの安心感を与えることができる。

# 解説系のチャンネルが伸びている理由
・政治・生活の不確実性が上がりきった結果、視聴者の一次ニーズが 「怒りの発散」 < 「不安の解消（判断軸・見通し）」 に寄りやすいからです。

・政治はコントロール不能と日本人全体が理解している
混沌としているからこそ“整理と判断材料、今後の解決策”に価値が出る。

・政治系視聴者の自己像・理想像である学んでる自分と相性が良い
落とし要素などの、エンタメ、ゴシップな内容を見てる自分より“学んでる自分”でいたい。
フロントの訴求が「完全解説/全史/なぜ」で中身でゴシップ、いじりなどの面白要素になっている方が自分の理想像を満たすためにもクリックしやすい。ただ、ゴシップ、エンタメ要素は需要があるため、中身で組み込む
→解説系をクリックする自分が自己像を満足させることができる。

# 政治系とエンタメ要素の動画編集の相性が良い理由
政治は重いテーマでもありつつ、歴史系と違って、過去の悲惨な出来事ではないので、ギャグ要素との相性が良いと言うことでしょうか？
そして、政治系は、深刻な問題で、重い空気で突っ込んでしまうと、視聴者が自分ごとかされる感が強く、不安を感じてしまい、安心感やエンタメ要素が減りストレスがかかるため、明るめの編集かつギャグ要素として触れるような面白要素、パチンコの編集のような過剰に派手な編集形態が相性が良いのでしょうか？

政治の重さ=現在進行・生活直結・対立が可視化されやすい＝自分事化/不安/怒りが発火しやすい重さ
つまり、政治の重さは「不安を増やす」方向に振れやすい→ 深刻なトーン一辺倒の編集だと、視聴者は“解決不能な現実”に直面して 疲れる/しんどい になりやすい

# 編集（派手/ガチャガチャ）が与える感覚

- **スカッと感（代弁カタルシス）**：ツッコミ/いじり/強い強調で「言ってくれた」を増幅
- **重要感/緊急感**：派手さ＝“今見ないと損”の圧を作る
- **退屈しない感（離脱防止）**：政治の重さをテンポで見せる（集中が戻る）
- **快感の連打**：驚き・笑い・強調を刻んで“見てて飽きない”を作る
- ※最終的に「整理→判断材料（学び）」で回収しないと、疲労/不信に振れやすい
