# サムネや訴求からどのような感覚を得ることができるか

・「なぜ〜なのか」とはきり分かりやすく簡潔に訴求していることから動画を見ることで、事象が理解できる裏切らない「安心感」「腹落ち感」を強調している
・赤色、黄色、黒などの危険性を煽るような色合いが使われることでの「重要感」
・サムネで訴求するワードが簡潔に訴求されている→「スッキリ感」「納得感」を感じる
・解像度が高く簡潔な訴求が多い→“視聴で解像度が上がる”期待感


- **編集（簡素/簡潔）が与える感覚（仮説）**
  - **知的没入感**：余計な刺激を減らし知識理解に集中させる
  - **落ち着き/安心感**：煽り過多に見せず「これを見れば分かる」雰囲気を作る
  - **信頼感（誠実さ）**：派手演出が少ないほど「冷静に説明してくれそう」に寄りやすい
  - **疲れにくい感**：重いテーマでも“頭が疲れない”体験で習慣化しやすい
- **道徳（視聴者の予測）を揺さぶる逆転ワード**：「今日から犯罪は合法」など（意外性/驚き→“理由が知りたい”）

# 世界史と簡潔な動画編集の相性が良い理由
世界史は、触れられてはいけないタブーの要素、過去の悲惨な出来事の要素もあるため、重い空気での解説が好まれるのでしょうか？

編集：世界史：出来事との距離がある（過去の出来事）ため、倫理/タブー/悲惨さ＝敬意/厳粛さが求められる
世界史は、触れられてはいけないタブーの要素、過去の悲惨な出来事の要素もあるため、重い空気での編集解説が好まれる。
「没入」と「信頼」が重要で、悲惨/タブー領域が多く、軽さが“不謹慎”に見えるリスクがある
