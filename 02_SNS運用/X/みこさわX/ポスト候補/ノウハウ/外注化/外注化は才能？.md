# 回答
人数に関しては上限値はあると考えます。
ただ伸ばすことは可能。

教えるというよりもその人が抱えられる人間の許容範囲があると考えます。

そしてやり方ではなくで仕組みで作業や管理を軽減していく。

外注化で必要なスキルは色々とありますが、

EQやタスク管理、実行力、リスク管理能力など様々です。

特に人間関係の部分なので、
EQは大事だと考えます。

あとは作業などのタスクを管理して仕組みで軽減していくノウハウやスキルなどでしょうか。

・タスク管理シート
・チェックシート
・添削AIなど

外注化を行い人材管理を行う際に

・どの作業を
・誰が
・どのくらいのスピードで
・どのくらいの質が保てているのか

を管理しないといけません。

この各作業の負担を軽減してあげる事は、
スキルの仕組み。

人の育成やマネージメントなどは育成能力（ティーチングやコーチング能力）や
EQの部分が大事だと考えています。

自分と他人の感情を察知・調整し、行動に結びつける能力。
→「人をどう動かすか・関係を保つか」の力。

これがEQです。

全員人間なので大なり小なりEQは持ち合わせています。
EQが低い人は上記の感情を察知して調整し行動に結びつける力が低いという事です。

逆に言えば、
外注さん（相手が）どのような感情でどのように思っているのか？これを察して
行動（オペレーション）して結果（仕組み化やAIでの効率化）に結び付けてあげる。

である程度はできます。

ただ、そこに多くの人数を抱えると人間的な摩擦やマルチタスクの処理能力なども関わってくるので、
その人が耐えられる限界値は存在すると考えます。

もちろん後天的に伸ばせる力ではありますが、
時間がかかるものです。

そして伸ばせてもどこまでも伸ばせるものでもなく、
個々の限界値はあります。

EQは、先天的なものも多少はあると考えます。

ただ実世界で通常の業務を行う程度のEQであれば多くの人間が持ち合わせていると考えます。

そしてその後の、知識や経験などの過去の体験でEQは形成される。

経験は人生を通した経験です。
子供のころから今までの経験値でEQは作られていく。

先天的な上限値は存在するが、
上限までは伸ばすことは可能。

上限値を上げるためには、
その人たちに過去にはない経験を入れていかないと
広がらないと考えます。

どちらというわけではなく、

双方がありその掛け算という考え方です。

センスは磨くことができる。

というのが美容師時代の考え方でやってきました。

そもそも論ですが、才能で全てが決まってしまうのであれば、
世の中は成り立ちません。

なぜなら、私たちのような通常の仕事【井上は元美容師】から
このYouTube業界に飛び込んで年商１０億レベルまで持っていけました。

これは後天的な力です。

過去の経験や今の経験をどのように捉えているのか。
【環境や他の人間、社会や自分など色々な観点から見る（メタ認知の力）】

そこを踏まえた上で何を良しとして、
どの様な選択肢を並べているのか。
【選択肢の幅は知識と経験】

そしてそれを選ぶ場合にはどんな基準で考えているのか
【メリットデメリットだけでなく、環境、自分、他者に対してどのような影響を与えるのかをメタ認知できる】

この無数の組み合わせで
相手が選ぶことが決まってきます。

なので、
向き不向きは多少はあるが、
アスリートや世界的な地位や名誉まで上り詰める、と言ったら才能×努力ですが、
一般的な業務においては努力でだいたいどうとでもなる。

というのが私の考えです。
