# 自分のビートで行け
発達障害の人の中には、子供の頃は変に大人びていたのに、大人になった途端子供っぽくなるという、現場でよく語られる症状があります。「発達の非対称性」です。一般的な子供は発言や言動が幼いですが、一部の発達障害傾向の子供は言葉や知識がやたら大人びている。そして大人になって、一般的には社会性や自制心が成熟していくのに対し、一部の発達障害傾向の人は「空気が読めない人」になる。

ただ、それは最初からそうだったと私は見ています。彼らは小学生の段階で政治や宇宙を語ることができ、知識や語彙が周りより早熟なんです。そう、知識や語彙だけが早熟。それは子供の時期には「大人みたいだね」と扱われますが、本当に大人になると「空気が読めない変な人」になる。でもそもそも、子供バイアスを抜けば、別に最初から空気は読んでいなかった。社会性の成長速度の遅さが、成長過程を経て露呈していっただけです。ただそれも人より遅いだけで、いずれできるようになる。発達障害は劣っているのではなく、普通の人とは違うビートで生きているだけなんです。

子供の頃はそれで褒められたから続けた。それが、ある時からそれだけでは褒められなくなった。どころか、「空気が読めない」と言われるようになった。それで自分が発達障害なんだと分かった。ただ、発達障害だから何もできないとかではなく、発達障害なりにズレながらも進み、ブレながらも学習していくことができるんです。

「私の人生は35から始まった」。
私の人生は35歳くらいから始まったので、40くらいになって普通の人が20で気づくことに気づいたりしています。エプロンをつけると料理中に油がハネても平気とか、アイロンをかけると服のシワが伸びて綺麗になるとか、寿司はうまいとか、ビールはうまいとか、全てにおいて20年遅い。

別の人もこう言ってました。「ドキドキだけが恋愛じゃないとか、高級な食べ物は美味しいとか、掃除はマメにしないとすぐ汚れるとか、普通の人より気づくのがめっちゃ遅かった。その時、人より成長が遅いことに落胆していたけど、気づき始めてから自分の人生が始まったって感覚が生まれてきた。よく人生は20代で決まると言われて焦っているけど、ゆっくり進もうと思うよ」と。

彼女は実際に35歳になって、自分が普通の人より少し遅れていることに気づいた。彼女も含め、大人になってみんなこれに気づくんです。「自分の人生は自分でデザインしていい」ってことに。

冒頭に喋った男性は、子供の頃から刻んだビートが大人になって他人と違うということに気づいた。そこから進んで、彼女はエプロンをつけてみたり、アイロンを当ててみたり、ビールを飲んでみたり。30を超えてから初めて他人のビートを真似てみたら、めちゃくちゃ生きやすいことに気づいた。みんな気づいていくんです。「あれ、こっちの方がもっと生きやすいじゃん」って。

実はこれは、発達の遅れとかそんな簡単な言葉では片付けられない話なんです。脳科学の世界では、前頭前野や計画、自分のコントロールをする場所は25から30歳で完成すると言われていますが、強いストレス下ではこの成熟がさらに遅れると言うんです。例えば日本は年功序列でレール型の社会なので、20代までは親と会社が作った物語に乗るのが精一杯。そしてその負荷やストレスが前頭前野の発達を遅らせるんです。

これが昇進や同級生との格差、出産や親の介護などが一気に押し寄せる35歳前後になって、人生で初めて自分で考えるしかない局面が来るんです。そして遅れて目覚める大半の人がそんな感じです。

仏教にはこういう言葉があります。「人は十分に苦しまないと世の中の構造を理解することができない」。冒頭の彼は自分が認められないどころか、周りから否定されていることに苦しんだ。そんな、自分で考えるしかない局面で自分が人と違うことに気づいた。自分の構造に気づき、そして世界の構造に気づいた。彼は20代前後で体験し気づいたので、世の中への理解に関しては人より早い可能性すらあります。

続いての彼女は、35歳という脳の成熟に伴い、唐突に自分がいかに致命的なことをしていたかあらゆることに気づき始めた。彼女は実は順当なんですね。痛みを体験しなければ成熟は少し遅れるが、遅いほど悟りは重みを持ちます。

多くの人は、自分にとって本当に大切だったことを時間差で遅れて気づくものなんです。そして「自分はもう取り返しがつかない存在になったのではないか」と思うこともあるんです。ただ、これは個人的感想ですが、時間は永遠のように非常に長く、20歳で気づいても35歳で気づいても、取り返しがつかないのは過去だけで、現在以降は全て挽回が可能なんです。

これは矛盾しているようですが、時間が足りないと思うこともある反面、時間が長すぎると思うこともあるんです。例えば、メロは個人でずっと生きているので、自分1人でお金を稼ぎ始めてから7年ぐらい経ったんですね。その間、言ってしまえば誰からも給料をもらっていないので、ざっと計算してもう6万1320時間は自分でやりくりしているんです。もう正直長いんですね。学生の時、なんとなく「このままじゃダメだな」って思って何かやろうとやり始めてから6万時間分経ったが、まだあと60年ぐらいは形を変えても続けなきゃいけないんで、時間算で52万5000時間分残ってる。ナイーブになって自信がなくなってくるとそういう発想しちゃうって話なんですが。

何が言いたいかって、早送りして何かを急いで学んだり、ゲームに課金して倍速でしたり、行きたいところに一辺に行ってみたり、効率的に量をこなして生きていると、もうすぐにやることがなくなって、時々気が滅入るくらい長いなって思うことがあるんです。やり尽くしたなって思っても、「まだまだ時間あるじゃん」って。「まだなのか」って。「これ以上なんかあんのかな」って無気力になってくる側面がある。

「気づくのが遅すぎて自分はもう取り返しのつかない存在じゃないか」って、全く問題ないんです。例えば、早く何かに気づいて早くそれができるようになって、その後どうなるかって、同じシーンを繰り返し見ることになるだけなんです。仮面ライダーとか水戸黄門みたいに毎回結論が同じ物語って、脳の発達が幼い子供とかおじいちゃんなら楽しめるけど、10代から60代くらいの大人にとっては、毎回同じ結論の物語を楽しめってのは結構きついものがあるんです。それに近いです。

「取り返しがつかないんじゃないか」って20代とか30代で気づいても、その人たちの残りの日中の活動時間って40年から30年残ってる。23万から17万時間です。もう1時間ですらまともに椅子に座っていられない人間にとって、その23万倍の23万時間というものは、途方もないくらい長いんです。ゆえに、何かを悟っても数万時間はほとんど堕落しているんです。

ゆえに、いつ気づいたか、その遅れを量る必要はないんです。いつ気づいたか、結局それも他人との比較であり、人生そのものの価値とは関係ない。メロは学生の頃、「この年になったら歴史に興味を持ち出すから、この年から歴史の勉強をしそう」って決めてた年齢があるんです。そこまでまだあと34年あるんですが、「知らないこととかも取っておくか」ってくらいのスピード感が自分にとってはマジでちょうどいいんですね。

結局最後は全てが収束していくので、その過程は自分の持ってるビートで進めばいいんです。
