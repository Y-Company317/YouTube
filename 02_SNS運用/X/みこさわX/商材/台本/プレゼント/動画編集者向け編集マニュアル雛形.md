# 🎬 YouTube動画編集完全マニュアル

## 🎯 はじめに：編集者の役割
編集者の仕事は、素材を繋ぐことではありません。
**「視聴者のストレスを極限まで減らし、動画の世界に没入させること」**です。
あなたの編集一つで、視聴維持率は10%以上変わります。

---

## ⚙️ 基本設定・納品形式
作業を開始する前に、以下の設定を確認してください。

| 項目 | 設定値 | 備考 |
| :--- | :--- | :--- |
| **解像度** | 1920 × 1080 (Full HD) | 4K素材の場合もこのサイズで書き出し |
| **フレームレート** | 30fps または 60fps | チャンネルの規定に従う |
| **書き出し形式** | .mp4 (H.264) | ビットレート：10〜15Mbps推奨 |
| **使用ソフト** | Premiere Pro / Final Cut Pro | 指定がある場合はそれに従う |

---

## ✂️ カット編集の基準（ジェットカット）
当チャンネルでは**「徹底的なジェットカット」**を採用しています。
「間」は視聴者の離脱ポイントです。

*   **無音部分の削除**: 波形を見て、無音部分（0.5秒以上）は全てカットしてください。
*   **「えー」「あー」の削除**: フィラー（言い淀み）は全て削除。
*   **ブレス（息継ぎ）の削除**: 大きな息継ぎ音はカット、または音量を下げてください。
*   **食い気味に繋ぐ**: 前の言葉の語尾と、次の言葉の出だしが被るギリギリまで詰めてください。

> 💡 **判断基準**
> 自分でプレビューを見て「一瞬でも退屈だ」「遅い」と感じたら、それは視聴者にとっては**「戻るボタンを押す理由」**になります。

## 📺 テロップ（字幕）のルール
視認性を最優先します。デザイン性よりも「読みやすさ」を重視してください。

### 1. フォント設定
*   **メインフォント**: ヒラギノ角ゴシック W8 / Noto Sans JP Bold
*   **禁止フォント**: 明朝体（細くて読みにくい）、ポップ体（安っぽくなる ※演出時は除く）

### 2. 配色ルール（チャンネル規定）
*   **強調（ポジティブ）**: 赤文字（#FF0000） + 白フチ
*   **強調（ネガティブ）**: 青文字（#0000FF） + 白フチ
*   **通常**: 白文字 + 黒フチ（不透明度80%程度の座布団推奨）

### 3. 表示位置
*   **セーフマージン**: 画面端から**10%以内**には配置しない（スマホのUIと被るため）。
*   **位置固定**: 基本的に下部中央で固定。話者が変わるごとに位置をズラさない。

---

## 🎵 BGM・SE（効果音）の挿入ルール
音は動画のクオリティを左右する「心臓部」です。

### 1. 音量バランス（目安）
*   **声（ボイス）**: 0dB 〜 -3dB（基準）
*   **BGM**: -20dB 〜 -25dB（声の邪魔をしないレベル）
*   **SE（効果音）**: -5dB 〜 -10dB（アクセントとして聞こえるレベル）

### 2. SEを入れるタイミング
*   **画面転換時**: スライド、ズームなどの動きに合わせて「シュッ」「パッ」などを入れる。
*   **感情表現時**: 驚き（ドーン！）、疑問（？）、注目（キラキラ）など。
*   **頻度**: **5秒〜10秒に1回**は何かしらの音が入っている状態を目指してください。

## 🖼 素材選定のルール
*   **商用利用可**: 必ず商用利用が許可されているサイトからダウンロードしてください。
    *   推奨: Unsplash, Pexels, いらすとや, MotionElements
*   **著作権への配慮**: テレビ番組のキャプチャ、芸能人の画像、他人のYouTube動画の無断使用は**絶対禁止**です。
*   **画質の確保**: 荒い画像、透かし（ウォーターマーク）が入っている画像は使用しないでください。

## ✅ 納品前セルフチェックリスト
書き出しを行う前に、以下の項目を必ず確認してください。

- [ ] **誤字脱字チェック**: テロップの誤字は致命的です。必ず2回見直してください。
- [ ] **音量バランス**: スマホのスピーカーで聞いて、声が聞き取りやすいか確認しましたか？
- [ ] **黒味の確認**: 動画の最初や最後、クリップの間に意図しない隙間（黒味）はありませんか？
- [ ] **レイヤー順序**: テロップが画像の下に隠れていませんか？
- [ ] **指定フォーマット**: ファイル名（例：`20240208_案件名_氏名.mp4`）は正しいですか？
- [ ] **プロジェクトファイルの整理**: 修正依頼があった際に、すぐに作業できる状態ですか？
