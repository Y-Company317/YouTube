# 低単価note サムネ文言選定書

> 参考デザイン：いるかチャンネル公式LINE特典サムネ（黒背景×ゴールド×いるかキャラ）

---

## 構成ルール

各サムネは以下の3層＋バッジ構造で統一する

| レイヤー | 役割 | フォントサイズ |
|---|---|---|
| 上段（フック） | 権威性・コンテキスト・好奇心喚起 | 小 |
| 中段（ベネフィット） | 具体的な成果・数字・メリット提示 | 中 |
| 下段（メインコピー） | 商品名・ツール名 | 大（太字） |
| バッジ | トレンド・網羅性・特別感 | 小（装飾付き） |
