# 第4章：YouTube×AI外注化という「最短ルート」

## 凡人の戦略「経営者視点」と「勝てるジャンル選定」

僕には、天才的な企画力も、圧倒的な編集スキルもありません。
だからこそ、凡人なりの戦略を立てました。

「自分で作業を巻き取りすぎない」

これです。

「仮説を持って動く」
「仕組みを作って回す」

経営者の視点です。

*   単純作業はAIに任せる。
*   思考を要する作業は、信頼できるディレクターに任せる。
*   僕は、全体のチャンネル設計と意思決定だけに集中する。

そしてもう一つ、重要なのが「勝てるジャンル選定」と「習慣」です。

1.  **「権威性」と「速報性」が不要なジャンルを選ぶ**
    *   医師や専門家が強いジャンル（権威性）は避ける。
    *   スポーツ速報など、常にPCに張り付く必要があるジャンル（速報性）は副業では無理。
    *   狙い目は、入念なリサーチと準備さえすれば非トレンドのコンテンツでも勝てるポジション。

2.  **競合分析を徹底する**
    *   毎朝、競合の動画をチェック。
    *   「テーマ」「サムネイル・タイトル」「再生数」の3点を分析。
    *   これを続けると、自然と伸びる企画・サムネの傾向が一発でわかる様になってきます。

3.  **「クオリティ」で差別化できるジャンルを狙う**
    *   再生数はあるが、動画の中身（台本や編集）が微妙なジャンルを探す。
    *   サムネの訴求力は真似しつつ、動画の中身を「テレビ番組（世界仰天ニュースなど）」のように面白くする。
    *   これだけで、後発でも十分に勝てます。

4.  **長期視点・一点突破でリソースを投下する**
    *   「3ヶ月で損切り」は早すぎる。僕は1年見る覚悟で挑みました。
    *   僕が始めて収益化まで行ったイジキタさんのチャンネルでは、収益化まで4ヶ月。
    *   その間、平日5時間、土日8時間、週40時間を一つのことに集中させました。
