# 自己紹介note構成案：偏差値30代の元ライターがYouTube×AIで1億円稼ぐまで

## タイトル案
*   **本命：** 偏差値30代・手取り17万の元ライターが、YouTube×AIで「1年で1億」稼いでFIREするまでの全記録
*   **対案A：** 【実話】「一生労働で終わる」と絶望した僕が、24歳で資産億越え・銀盾4枚を手にするまで
*   **対案B：** 才能なし・金なし・学歴なし。偏差値30代の凡人が「YouTube×AI外注化」で人生をバグらせた話

## 全体想定文字数：8,000〜12,000文字

## 構成プロット

### 序章：現在の地点と、かつての自分 [1,000〜1,500文字]
*   **現在のステータス（権威性）**
    *   20代でYouTube収益1億円突破
    *   銀盾4枚保有、最高月商2000万円
    *   フル外注で10ch以上を運営し、場所と時間に縛られない生活
    *   **最大の武器：** 「アドセンス収益」と「バックエンド販売」の両立を実現する非属人最強のチーム構築
*   **過去の自分（共感・マイナスからのスタート）**
    *   青森の田舎出身、**偏差値30代のFラン大学卒**
    *   野球しかしてこなかった脳みそ筋肉質（刺激がないと死ぬ、成長が止まるのが怖い）
    *   手取り17万円の平凡な会社員
*   **フック（読者への問いかけ）**
    *   「特別な才能があったんでしょ？」→ 答えはNO。
    *   僕が持っていたのは、**「このまま一生、会社に飼われて終わるのか？」という強烈な恐怖**だけだった。

### 第1章：原体験という名の「呪い」と「渇望」 [1,500〜2,000文字]
*   **母の背中と誓い**
    *   シングルマザーの母。毎晩21時に帰宅し、泥のように眠る日々。
    *   「趣味も楽しみもなく、ただ生きるために働く」母を見て抱いた違和感。
*   **野球部時代の狂気と喪失（KGOより追加）**
    *   男子校・工業高校の体育会系野球部。理不尽な上下関係と罵倒の日々。
    *   **失ったもの：** 普通の青春（放課後の遊び、恋愛、行事）。
    *   **得たもの：** 異常な精神的体力と、引退時の「全能感（歓喜）」。
    *   **気づき：** あの「歓喜」をもう一度味わいたい。だから僕は今も仕事に狂っている。
*   **「普通の幸せ」への絶望**
    *   大学時代のバイト生活で感じた「生殺与奪の権を他人に握られる」屈辱。
    *   「普通の会社員＝精神的にも経済的にも豊かになれない」という確信。

### 第2章：泥臭い副業時代と「労働の壁」 [1,500〜2,000文字]
*   **孤独な闘争（KGOより追加）**
    *   **期間：** 2023年4月〜2024年9月。土日も含め、遊んだ記憶が一切ない。
    *   **行動：** 友人、恋人、家族との時間をすべて遮断。意図的に孤独を作り出した。
    *   **動機：** 「とにかく金が欲しい」という執着と、競合を見下し「絶対に負けない」という闘争心。自己洗脳に近い状態でPCに向かっていた。
*   **Webライターへの挑戦**
    *   ビジネス書100冊読んで気づいた「行動しなきゃゴミ」という真実。
    *   「書くこと」への憧れと、趣味（健康・食）を金に変える楽しさ。
    *   **ライターとしての気づき：** 「言語化能力」と「論理的思考力」は武器になる。しかし、労働集約型からは抜け出せない。
*   **ぶち当たった「限界」**
    *   本業180時間＋副業90時間。起きている時間はすべて仕事。
    *   それでも「月15万円」のアッパーが見えてしまった。
    *   ITエンジニアとしての本業も、天才たちには勝てないという挫折。

### 第3章：転機と脅威（YouTubeとの出会い＆AIの襲来） [1,000〜1,500文字]
*   **衝撃の出会い**
    *   ある健康系chのディレクター（当時24歳）との出会い。
    *   彼は副業でさらりと「月25万円」を稼いでいた。
    *   **気づき：** 能力の差ではない。「選んだ場所（乗り物）」の差だ。
    *   **伏線回収：** 実はこの彼とは後に再会し、今は僕の会社で一緒に大きな事業を動かす最強のパートナーになっています。
*   **ディレクター転向後の苦戦（KGOより追加）**
    *   2023年7月〜12月、半年間収益ゼロ。チャンネル打ち切り寸前。
    *   それでも諦めなかったのは、野球部時代に培った「理不尽への耐性」があったから。
*   **ChatGPTという黒船（2023年4月）**
    *   ライターとして積み上げたスキルが、AIに一瞬で代替される恐怖。
    *   逆に言えば、「AIを使役する側」に回れば勝てるという確信。

### 第4章：YouTube×AI外注化という「最短ルート」 [1,500〜2,000文字]
*   **爆発的成長**
    *   2024年1月、ついに収益化達成。
    *   4月：副業月収50万、5月：100万、7月：**730万**達成。
    *   **達成感：** オフ会でチヤホヤされる優越感。「俺は勝ったんだ」という確信。
*   **凡人の戦略**
    *   天才的なクリエイターにはなれない。だから「経営者視点」を持った。
    *   AIに作業をさせ、人に任せる（外注化）。
    *   **思考の転換：** 「どうしたらいいですか？」と聞く作業者から、「仮説を持って動く」経営者へ。

### 第5章：1億円稼いで見えた景色 [1,000〜1,500文字]
*   **手に入れたもの**
    *   ポルシェ、タワマン、ハイブランド…かつて憧れた物質的な豊かさ。
    *   しかし、それ以上に価値があったのは**「自由」**。
    *   誰にも命令されない。好きな時に起き、好きな場所で、好きな人とだけ関わる権利。
    *   母を見て抱いた「未来への恐怖」が消滅した瞬間。
*   **失ったものへの後悔はない（KGOより追加）**
    *   青春も、人間関係も犠牲にした。
    *   でも、その対価として得た「自信」と「自由」は、何物にも代えがたい。

### 第6章：あなたに伝えたいこと（現在・未来・ヴィジョン） [1,000〜1,500文字]
*   **僕の信条（KGOより追加）**
    *   **未来は自分が作る。**
    *   「自由な心で、自由な発想をし、自由な行動」を体現する。
    *   常識人ではなく、**「社会不適合者」**として自由を謳歌し、人々の行動を促す存在でありたい。
*   **現在の活動と未来の接続**
    *   **現在：** 時間と場所を選ばず、初期投資0円で始められる「YouTube」という武器を配っている。
    *   **未来：** 環境や判断の誤りで制限を受けている人々の「制限の解放」を実現する。
*   **みこさわチームの強み**
    *   ただのアドセンス狩りではない。
    *   **「アドセンス収益（フロー）」×「バックエンド販売（ストック）」の両立**が可能。
    *   属人YouTuberにはできない、組織的な「視聴者理解」と「教育」の仕組みがあるからこそ、爆発的な収益と安定を両立できる。
